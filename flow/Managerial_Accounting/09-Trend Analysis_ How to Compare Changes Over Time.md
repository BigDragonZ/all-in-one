Trend analysis is based on the idea that numbers change over time. We want to quantify this change, whether it's an increase or a decrease. There are a couple of ways to show this, and I'll demonstrate both. In our example, we have data for Year 1, Year 2, and Year 3. Revenue increased from 120 to 127, and then to 144. How do we calculate this trend?

One way is to express each year's value as a percentage of the base year (Year 1). For Year 1, we take 120 divided by the original 120, which is 100%. In Excel, you would make the denominator an absolute reference (using F4) so it always divides by the base year's value. When we copy this formula across, we see the trend goes from 100% in Year 1, to 105.8% in Year 2, and 120% in Year 3.

Alternatively, you can show only the percentage change from the base year. This is done by subtracting 1 (or 100%) from the calculated percentage. For the base year, the change is 0%. For the following years, you would show a 5.8% increase and a 20% increase, respectively. This is similar to how a consumer price index works, where the base year is set to 100, and subsequent years show the cumulative inflation (e.g., 105 for 5% inflation). For simplicity, we'll show the percentage change.

Let's apply this to all our data and analyze the trends:
- **Revenue:** Increased by 20% over two years. This is a positive trend, so we'll label it as **favorable**.
- **Cost of Goods Sold:** This is a cost, and it increased by 30%. Since the cost grew faster than revenue (20%), this is an **unfavorable** trend.
- **Net Income:** Increased by only 8.3%. While any increase is good, it's significantly lower than the 20% revenue growth. This indicates that rising costs are eroding profits, making this trend **unfavorable**.

This is a fundamental application of financial statement analysis. We use trend analysis to identify growth or reduction in key metrics. The same principle applies to decreases; for example, if a value went down, you would see a negative percentage change. This method works for both positive and negative trends.
