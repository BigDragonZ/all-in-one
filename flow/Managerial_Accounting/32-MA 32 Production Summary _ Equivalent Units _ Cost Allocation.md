### 初始数据与背景

我们来看一下四月份的成本数据。我们有期初在产品（Beginning Work in Process）的材料成本和转换成本。**转换成本（Conversion Costs）** 指的是人工和制造费用。如果制造费用是按直接人工成本分配的，那么它们完成的百分比是相同的，因此我们可以将这两项合并为转换成本。所以，我们将成本分为**直接材料（Direct Materials）**和**转换成本**两类进行分配。

在单位方面，我们有期初在产品 200 个，它们是部分完成的。本月我们新投入生产 3,700 个单位，期末在产品有 380 个。对于期末在产品，材料的完工进度是 55%，转换成本的完工进度是 45%。我们将使用**加权平均法（Weighted-Average Method）**，这意味着我们只关注已完工单位和期末在产品。

### 实体单位流转

首先，我们需要计算实体单位的数量。计算公式是：期初单位 + 本期投入单位 = 可供加工单位；可供加工单位 - 已完工单位 = 期末单位。我们已知其中的三个数据，可以推算出其余的。

- **期初在产品单位**：200
- **本期投入单位**：3,700
- **可供加工单位**：200 + 3,700 = 3,900

我们不知道完成了多少单位，但我们知道：

- **期末在产品单位**：380

因此，**本期完工并转出单位** = 3,900 - 380 = 3,520。
后续的所有计算都建立在这些实体单位数量的基础上。

### 第一步：汇总待分配的成本

在第一部分，我们汇总所有需要分配的制造总成本。

- **期初在产品成本**：$1,800 (材料) + $2,100 (转换) = $3,900
- **本期投入的直接材料成本**：$36,400
- **本期投入的转换成本**：$44,800

**待分配的总制造成本** = $3,900 + $36,400 + $44,800 = **$85,100**。
这个总数将是我们最后一步用来核对的数字。

### 第二步：计算约当产量（Equivalent Units）

接下来，我们计算约当产量。在加权平均法下，我们只考虑**已完工单位**和**期末在产品**。

- **已完工单位**：3,520 个。这些单位对于材料和转换成本都是 100% 完成的，所以它们的约当产量就是 3,520。
- **期末在产品**：380 个。
    - **材料的约当产量**：380 单位 × 55% = 209 单位
    - **转换成本的约当产量**：380 单位 × 45% = 171 单位

现在，我们将它们相加得到总的约当产量：

- **材料的总约当产量**：3,520 + 209 = **3,729**
- **转换成本的总约当产量**：3,520 + 171 = **3,691**

能否正确计算约当产量是掌握分步成本法的关键。

### 第三步：计算单位约当成本

在第三部分，我们将第一步的总成本除以第二步的约当产量，来计算单位成本。

- **单位材料成本**：
  (期初材料成本 + 本期材料成本) / 材料约当产量
  = ($1,800 + $36,400) / 3,729
  = $38,200 / 3,729 ≈ **$10.2440**

- **单位转换成本**：
  (期初转换成本 + 本期转换成本) / 转换成本约当产量
  = ($2,100 + $44,800) / 3,691
  = $46,900 / 3,691 ≈ **$12.7066**

为了避免舍入误差，建议至少保留四位小数。

- **总单位成本**：$10.2440 + $12.7066 = **$22.9506**

### 第四步：成本分配

最后，我们将总成本分配给完工产品和期末在产品。

- **完工产品成本（Cost of Goods Manufactured）**：
  完工单位 × 总单位成本
  = 3,520 × $22.9506 ≈ **$80,786**

- **期末在产品成本**：
    - **材料成本**：期末在产品材料约当产量 × 单位材料成本
      = 209 × $10.2440 ≈ $2,141
    - **转换成本**：期末在产品转换约当产量 × 单位转换成本
      = 171 × $12.7066 ≈ $2,173
    - **期末在产品总成本**：$2,141 + $2,173 = **$4,314**

### 成本核对

现在我们进行核对。我们将分配出去的成本相加：
完工产品成本 + 期末在产品成本 = $80,786 + $4,314 = **$85,100**。
这个结果与第一步中计算出的待分配总成本完全一致。这表明我们的计算是正确的。
