大家好，欢迎回来。今天我们继续学习管理会计的完整课程，本章的主题是相关成本（Relevant Costing）。这个概念也可能被称为增量成本（Incremental Costing）或差异成本（Differential Costing）等。本节我们将讨论一个特殊的订单决策问题。如果你是第一次来到这个频道，我叫 Jeff，我教授许多会计和金融课程。请点赞、评论、订阅，让我们开始学习相关成本的概念。

相关成本，也称为增量成本、差异成本或边际成本，核心在于研究当我们选择一个备选方案而非另一个时，哪些成本会发生变化。我们的基本假设是，总固定成本保持不变，而总可变成本会随之改变。这里的难点在于，问题有时会设置得让你误以为下一个产品的成本是某个特定数额，但你不能将固定成本计算在内。因此，你必须非常清楚地辨别什么是可变成本，什么是固定成本，以及哪些成本会随着决策改变，哪些不会。

在相关成本这个主题下，主要有三种典型问题：
- **特殊订单（Special Order）**：这是我们本视频要讲的。
- **自制或外购（Make or Buy）**：即外包决策。
- **保留或放弃产品线（Dropping a Product Line）**。

现在，让我们来看一个特殊订单的具体案例。假设 Scotch 公司有如下信息，而 Bonico 公司提出了一项特殊订单，希望以一个特殊价格大量采购。这就是特殊订单的定义：以一个更低的特殊价格，购买大量的商品。

这是我们目前掌握的情况：公司的产能是 8,000 单位，而目前的产量是 6,000 单位。特殊订单决策的一个前提条件是必须有**闲置产能**。我们有 2,000 单位的闲置产能，这意味着我们可以生产更多产品，这为我们接受特殊订单提供了机会。如果我们公司的产能是 8,000 单位，并且已经销售了 8,000 单位，那么我们就不可能以一个特殊低价去接受特殊订单，因为我们已经满负荷生产了。但正因为有闲置产能，我们才愿意考虑接受一份特殊订单，以一个特殊价格销售那些我们原本可能卖不出去的产品。目前，我们以每件 70 美元的价格销售 6,000 件产品。

以下是我们的制造成本信息。我们有材料、人工和制造费用。材料是可变成本，人工是可变成本，而制造费用既有可变部分，也有固定部分。固定制造费用总额为 72,600 美元。这时，Bonico 公司提出了一个特殊订单：他们想购买 1,000 单位产品，但希望价格是每单位 60 美元。我们是否应该接受呢？如果我们能从中获利，那么就应该接受；如果会亏损，那就不应该。

首先，我们来回答几个问题。**当前的单位成本是多少？**
我们的单位可变成本包括三项：$21.40 (材料) + $17.30 (人工) + $14.10 (可变制造费用) = **$52.80**。
接下来，单位固定制造费用是多少？我们将总固定费用 $72,600 除以当前产量 6,000 单位，得到 **$12.10**。
因此，当前的**总单位成本**是 $52.80 + $12.10 = **$64.90**。

这就是决策中容易犯错的地方。你可能会想：
> 既然我的成本是 64.90 美元，而对方只出价 60 美元，我肯定会亏钱，所以不能接受。

这种想法是错误的。固定成本的定义是什么？**固定成本是在一定的业务量范围内，其总额不随业务量变动而变动的成本。** 如果你认为下一件产品的成本仍然是 64.90 美元，那你就把每单位 12.10 美元的固定成本也当成了可变成本，这是不正确的。实际上，多生产一件产品的**增量成本**仅仅是可变成本，也就是 **52.80 美元**。

我们可以用成本公式 `y = a + bx` 来表示，其中 `y` 是总成本，`a` 是总固定成本，`b` 是单位可变成本，`x` 是产量。在这个案例中，成本公式是：
`y = 72,600 + 52.80x`
这意味着，我们已经承担了 72,600 美元的固定成本，无论我们是否接受这个新订单。因此，额外生产的每一件产品的成本只是 52.80 美元。虽然我们计算出当前平均成本是 64.90 美元，但这并不意味着下一件产品的成本也是 64.90 美元。

**我们当前的利润或亏损是多少？**
我们销售 6,000 单位，每单位售价 70 美元，成本 64.90 美元。
所以，当前总利润 = 6,000 * ($70.00 - $64.90) = **$30,600**。

**那么，我们应该接受这个特殊订单吗？**
很多人会直觉地用 64.90 美元的总成本去比较 60 美元的售价，然后得出结论：每卖一件亏 4.90 美元，总共亏损 4,900 美元，所以不能接受。但这是错误的。
正确的分析是，我们只考虑**增量**的收入和成本。
- 增量收入：1,000 单位 * $60/单位 = $60,000
- 增量成本（仅可变成本）：1,000 单位 * $52.80/单位 = $52,800
- **特殊订单带来的增量利润** = $60,000 - $52,800 = **$7,200**

由于我们能从这个特殊订单中获得 7,200 美元的额外利润，所以我们**应该接受**。

为了让你更清楚地理解，我们可以用一个简单的损益表来证明这一点。
| | **现状 (6,000 单位)** | **特殊订单 (1,000 单位)** | **接受订单后 (7,000 单位)** |
| :--- | :--- | :--- | :--- |
| 销售额 | $420,000 | $60,000 | $480,000 |
| 减：可变成本 | $316,800 | $52,800 | $369,600 |
| **贡献边际** | **$103,200** | **$7,200** | **$110,400** |
| 减：固定成本 | $72,600 | $0 | $72,600 |
| **净利润** | **$30,600** | **$7,200** | **$37,800** |

从上表可以看出，特殊订单本身会带来 7,200 美元的贡献边际。由于我们有闲置产能，接受这个订单**不会增加任何额外的固定成本**（例如，厂房租金、管理人员工资等都不会因此改变）。因此，这 7,200 美元的贡献边际将全部转化为净利润。我们公司的总利润将从 30,600 美元增加到 37,800 美元。

所以，尽管特殊订单的售价（60美元）低于我们当前的平均总成本（64.90美元），但由于固定成本不会改变，只要售价高于单位可变成本（52.80美元），这个订单就是有利可图的，能够增加公司的总利润。
