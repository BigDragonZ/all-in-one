Hello and welcome. My name is Jeff. Today we're working in managerial accounting, specifically the activity-based costing chapter. We're going to show an example of how to calculate the cost per unit using ABC (Activity-Based Costing).

The **Uniform Company** uses ABC and has the following predetermined activity rates:
- **Setup**: $2,500 per setup
- **Materials Handling**: $180 per requisition
- **Inspection**: $6 per unit inspected

The company produced 2,000 units. This production run had the following actual activity:
- **Setups**: 4
- **Material Requisitions**: 30
- **Units Inspected**: 2,000

The question is: what is our overhead cost per unit for these 2,000 units?

To solve this, we'll set up a chart to calculate the total cost allocated to this production run. We will multiply the actual activity level for each category by its predetermined activity rate.

- **Setup Cost**: 4 setups × $2,500 per setup = $10,000
- **Materials Handling Cost**: 30 requisitions × $180 per requisition = $5,400
- **Inspection Cost**: 2,000 units inspected × $6 per unit = $12,000

Next, we sum these costs to find the total allocated overhead.
Total Allocated Cost = $10,000 + $5,400 + $12,000 = $27,400.

Finally, to find the manufacturing overhead cost per unit, we divide the total allocated cost by the number of units produced.
$27,400 / 2,000 units = **$13.70 per unit**.

In summary, with Activity-Based Costing, we identify activities, determine an activity rate for each, and count the number of activities used. We then multiply these figures to allocate costs. By dividing the total allocated cost by the number of units, we find the overhead cost per unit. For this problem, it cost us **$13.70** of overhead per unit.
