### 什么是弹性？

在本次视频中，我们想讨论一个叫做“弹性” (elasticity) 的概念。这是一种让供需模型变得更有用的方法。到目前为止，当我们讨论需求和供给时，我们画出需求曲线和供给曲线，观察它们的交点，并知道这个交点告诉我们市场中最终形成的价格和商品交易的数量。但我们并没有真正关注如何绘制这些需求和供给曲线。在本章中，我们将深入思考这一点，这个概念将贯穿整个微观经济学原理课程的其余部分。

弹性是一个应用非常广泛的微观经济学概念。在我们深入探讨之前，我们首先需要思考我们想要衡量什么，以及如何衡量它。我们将要思考的这个度量，我们称之为 **需求价格弹性** (price elasticity of demand)。

### 弹性的直观理解

在介绍具体定义之前，让我给你一个关于弹性的直观解释。弹性并非经济学家凭空创造的概念，它本质上是一种衡量两个相关事物之间关系强弱的数学方法。我们可以在科学、物理或工程背景下讨论弹性，当然也可以在经济学背景下讨论。

让我举几个例子。假设你有一个橡胶球，你把它扔向墙壁，然后测量球撞墙后反弹滚动的距离。你扔球所用的力与它滚回的距离是相关的。在其他条件相同的情况下，你扔得越用力，球滚回的距离就越远。这还取决于球的材质。如果你扔一个棒球，它会反弹回来，但回滚的距离会比你扔一个“超级球” (super ball) 要短。用同样的力，超级球会滚得更远。因此，力与距离之间的关系，会因球的材质不同而有所差异。

再举一个例子。假设你在开车时踩下刹车踏板，然后观察汽车减速的快慢。如果你开过不同的车，你就会知道不同汽车的刹车响应程度是不同的。有些车，你轻轻一踩刹车，它就会迅速制动；而另一些车，你需要很用力地踩下踏板才能让它开始减速。这取决于很多因素，比如刹车系统的类型、刹车片的磨损程度等等。但这里有两个相关的变量：你踩下踏板的压力大小，以及汽车停止的速度。

### 需求价格弹性：定义与术语

在经济学中，我们感兴趣的是商品价格与其需求量之间的关系。我们知道，需求曲线是基于需求表（demand schedule）绘制的，需求表列出了一系列价格以及在每个价格下消费者愿意购买的数量。根据需求定律，当价格上涨时，需求量会下降。但我们更关心的是，需求量到底下降了多少？

如果价格上涨，需求量是急剧下降，还是仅仅略有下降？思考一下你购买的两种商品：汽油和百事可乐。

假设你看完这个视频后去加油，结果惊讶地发现油价是你预期的两倍。你可能会感到失望，因为现在每加仑汽油的成本翻倍了。你可能会想办法少用点油，比如减少去商店的次数，或者一次性买更多的东西。但你仍然需要开车去上课、上班，所以即使你可能会少用一点油，但不会少用很多。在这种情况下，价格大幅上涨，但你的需求量响应不大。

现在，考虑你对百事可乐的需求。假设只有百事可乐的价格翻了一番。你会如何应对？对很多人来说，如果百事可乐价格翻倍，他们会直接转向可口可乐或其他软饮料。在这种情况下，一个相对较小的价格上涨，就会导致人们的行为发生巨大变化。

汽油价格可以大幅变动而人们的响应不大，而百事可乐价格稍有变动人们就会做出巨大响应。这两种商品的响应度是不同的。你可能已经猜到了，这两种情况的主要区别在于，百事可乐有很好的替代品（可口可乐），而汽油则没有。

现在，让我们给出需求价格弹性的正式定义。

**需求价格弹性 (Price Elasticity of Demand, PED)** 衡量的是需求量对价格变化的响应程度。

经济学家使用两个术语来描述这种响应程度：
- **弹性 (elastic)**: 当需求量对价格变化响应很大时。
- **非弹性 (inelastic)**: 当需求量对价格变化响应很小时。

你可以把这想象成一根橡皮筋。如果它能拉伸很多，我们就说它是有弹性的；如果它不怎么能拉伸，我们就说它是非弹性的。从本质上讲，弹性衡量的是当一种商品价格上涨时，消费者愿意放弃该商品的程度。

### 影响需求弹性的因素

有几个因素决定了需求弹性的大小：

1.  **可替代品的可用性 (Availability of Close Substitutes)**
    -   如果一种商品有许多近似的替代品，那么它的需求就是 **弹性** 的。例如，百事可乐和可口可乐对许多人来说几乎是完美的替代品。如果百事可乐价格上涨，人们会轻易地转向可口可乐。
    -   如果一种商品几乎没有好的替代品，那么它的需求就是 **非弹性** 的。例如，汽油。虽然步行、滑板或骑马是替代方案，但对于长距离出行来说，它们都不是好的替代品。因此，即使油价上涨，人们也很难大幅减少汽油消费。

2.  **时间跨度 (Passage of Time)**
    -   在 **短期** 内，需求往往是 **非弹性** 的。如果今天油价翻倍，你可能可以采取一些措施，如拼车或减少不必要的出行，但你仍然需要去上班上学，所以你的消费量不会立即大幅下降。
    -   在 **长期** 内，需求会变得更具 **弹性**。如果高油价持续数月，你可能会考虑换一辆更省油的汽车，甚至换成摩托车。你有更多的时间来调整你的消费行为。

3.  **必需品 vs. 奢侈品 (Necessity vs. Luxury)**
    -   **必需品 (necessities)** 的需求往往是 **非弹性** 的。例如食物和水，因为它们没有好的替代品，所以即使价格上涨，你也必须购买。
    -   **奢侈品 (luxuries)** 的需求往往是 **弹性** 的。例如看电影。如果电影票价突然上涨，你可以轻易地选择其他娱乐活动，比如去听音乐会或做其他上百种周末可以做的事情。

4.  **市场的定义 (Definition of the Market)**
    -   **宽泛定义的市场** (broadly defined market) 的需求是 **非弹性** 的。例如“蔬菜”市场。蔬菜的替代品是肉类、谷物等非蔬菜食品，替代品范围有限。
    -   **狭窄定义的市场** (narrowly defined market) 的需求是 **弹性** 的。例如“胡萝卜”市场。胡萝卜的替代品包括所有其他蔬菜以及所有非蔬菜食品，替代品非常多。因此，市场定义得越窄，替代品越多，需求弹性就越大。

5.  **商品在消费者预算中所占的份额 (Share of the Good in the Consumer's Budget)**
    -   如果一种商品在你的预算中只占很小一部分，其需求往往是 **非弹性** 的。例如口香糖。你可能根本不会注意它的价格变化，因为即使价格上涨一点，对你的总支出影响也很小。
    -   如果一种商品在你的预算中占有较大份额，其需求则更具 **弹性**。例如汽油或房租，你会对它们的价格变化更加敏感。

### 如何计算需求价格弹性

计算需求价格弹性是一个非常简单的数学过程。如果你能计算两个数的平均值，那么最难的部分你已经掌握了。

需求价格弹性的定义公式是：

**PED = 需求量变化的百分比 (%ΔQd) / 价格变化的百分比 (%ΔP)**

我将使用希腊字母 Delta (Δ) 来表示“变化”，所以公式可以简写为：

`PED = %ΔQd / %ΔP`

**一个简单的例子：**
假设价格上涨了 10%，导致需求量下降了 20%。
`PED = (-20%) / (10%) = -2`

由于价格和需求量总是呈反向关系（一个上升，另一个下降），所以需求价格弹性的计算结果总是负数。因此，经济学家通常会忽略负号，直接使用其绝对值。在这个例子中，我们会说弹性是 2。

### 中点法 (The Midpoint Method)

在计算百分比变化时，我们面临一个问题：结果会因你选择的起点不同而不同。

例如，假设你的两次考试成绩分别是 80 分和 100 分。
-   从 80 分到 100 分：变化量是 20。百分比变化是 `(20 / 80) * 100% = 25%`。
-   从 100 分到 80 分：变化量是 20。百分比变化是 `(20 / 100) * 100% = 20%`。

使用相同的两个数字，我们得到了不同的百分比变化。为了消除这种不一致性，经济学家使用 **中点法** 来计算百分比变化。我们不再用初始值作为分母，而是用两个值的 **平均值（或中点）** 作为分母。

使用中点法，上述例子的百分比变化将是：
`变化量 / 平均值 = 20 / ((80 + 100) / 2) = 20 / 90`
无论你是从 80 到 100，还是从 100 到 80，用中点法计算出的百分比变化都是相同的。

因此，使用中点法的弹性计算公式可以分解为：
-   **%ΔQd = (数量变化量 / 平均数量) * 100%**
-   **%ΔP = (价格变化量 / 平均价格) * 100%**

最终的弹性公式是：
**PED = (数量变化量 / 平均数量) / (价格变化量 / 平均价格)**

### 计算示例

**示例 1：比尔的披萨需求**
-   当披萨价格为 10 美元时，比尔每年吃 50 个。
-   当价格降至 8 美元时，他每年吃 70 个。

1.  **计算数量的百分比变化 (%ΔQd):**
    -   数量变化量 = 70 - 50 = 20
    -   平均数量 = (50 + 70) / 2 = 60
    -   %ΔQd = 20 / 60 = 1/3

2.  **计算价格的百分比变化 (%ΔP):**
    -   价格变化量 = 10 - 8 = 2
    -   平均价格 = (10 + 8) / 2 = 9
    -   %ΔP = 2 / 9

3.  **计算需求价格弹性 (PED):**
    -   PED = (%ΔQd) / (%ΔP) = (1/3) / (2/9)
    -   PED = (1/3) * (9/2) = 9/6 = 3/2 = 1.5

**示例 2：**
-   当价格 (P1) = 2 美元时，数量 (Q1) = 13。
-   当价格 (P2) = 10 美元时，数量 (Q2) = 3。

1.  **计算 %ΔQd:**
    -   数量变化量 = 13 - 3 = 10
    -   平均数量 = (13 + 3) / 2 = 8
    -   %ΔQd = 10 / 8 = 5/4

2.  **计算 %ΔP:**
    -   价格变化量 = 10 - 2 = 8
    -   平均价格 = (10 + 2) / 2 = 6
    -   %ΔP = 8 / 6 = 4/3

3.  **计算 PED:**
    -   PED = (%ΔQd) / (%ΔP) = (5/4) / (4/3)
    -   PED = (5/4) * (3/4) = 15/16

### 教学方法的重要性

我过去常常直接向学生展示一个复杂的、合并在一起的弹性公式，结果发现很多学生都难以掌握。这个公式看起来是这样的：
`PED = (Q1 - Q2) / ((Q1 + Q2)/2) / (P1 - P2) / ((P1 + P2)/2)`

这个公式在数学上是正确的，但它看起来非常吓人，会让很多学生望而却步。后来，我改变了教学方法，将计算过程分解成几个简单的步骤：首先计算数量的百分比变化，然后计算价格的百分比变化，最后再将两者相除。这种方法让学生觉得简单得多，也更容易理解。所以，如果你在教科书或其他视频中看到那个复杂的公式，不要害怕。它和我在这里展示的分步方法是完全一样的，只是后者更容易理解和操作。

### 如何解读弹性值

计算出的弹性数值告诉我们需求的具体响应程度。

-   如果 **PED > 1**，我们说需求是 **弹性 (elastic)** 的。
    -   这意味着需求量变化的百分比大于价格变化的百分比。消费者对价格变化非常敏感。
-   如果 **PED < 1**，我们说需求是 **非弹性 (inelastic)** 的。
    -   这意味着需求量变化的百分比小于价格变化的百分比。消费者对价格变化不敏感。
-   如果 **PED = 1**，我们说需求是 **单位弹性 (unit elastic)** 的。
    -   这意味着需求量变化的百分比恰好等于价格变化的百分比。

**如何用语言描述弹性值？**
弹性值本身代表了价格每变化 1%，需求量会变化多少个百分点。

-   如果 PED = 2，你可以说：“价格每变化 1%，需求量就会反向变化 2%。”
-   如果 PED = 15/16，你可以说：“价格每变化 1%，需求量就会反向变化 15/16%。”
-   如果 PED = 4/3，你可以说：“价格每变化 1%，需求量就会反向变化 4/3%。”
