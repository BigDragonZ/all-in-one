## 四种市场结构的比较

我们来讨论第四种市场类型：寡头垄断（Oligopoly）。寡头垄断意味着市场由少数几家公司主导。任何超过一家公司（垄断）的市场都可以被视为寡头垄断。

为了更好地理解，我们回顾一下已经讨论过的市场结构。我们有一个谱系，一端是**完全竞争 (Perfect Competition)**，另一端是**垄断 (Monopoly)**。我们还讨论了**垄断竞争 (Monopolistic Competition)**，它基本上是产品存在差异化的完全竞争。现在，我们将要讨论的寡头垄断，则更接近于垄断这一端。一旦市场中不止一家公司，它就成为寡头垄断；随着公司数量的增加，它会逐渐向垄断竞争和完全竞争过渡。

让我们来比较一下这些不同市场类型的结果：
- **完全竞争**:
    - 价格等于边际成本 (P = MC)。
    - 价格被推至平均总成本的最低点 (P = min ATC)。
    - 利润最终为零 (Profit = 0)。
    - 实现了生产效率和消费效率，无谓损失为零 (Deadweight Loss = 0)。
- **垄断**:
    - 价格高于边际成本 (P > MC)。
    - 价格不会被推至平均总成本的最低点。
    - 只要产品需求强劲，长期利润可以为正 (Profit > 0)。
    - 存在无谓损失 (Deadweight Loss > 0)。
- **垄断竞争**:
    - 价格高于边际成本 (P > MC)，这一点与垄断相似。
    - 价格等于平均总成本 (P = ATC)，但不是最低点，公司存在过剩产能。
    - 由于自由进入，长期利润为零 (Profit = 0)，这一点与完全竞争相似。
    - 存在无谓损失 (Deadweight Loss > 0)。但由于利润为零，监管很难实施。其优点是产品种类繁多，质量不断提升。

## 寡头垄断的特征与分析方法

对于寡头垄断，其结果在很多方面与垄断相似：
- 价格通常高于边际成本 (P > MC)。
- 存在进入壁垒，因此公司可以在长期内获得正利润 (Profit > 0)。
- 存在无谓损失 (Deadweight Loss > 0)。

所有公司都通过生产边际收益等于边际成本 (MR = MC) 的数量来实现利润最大化。然而，分析寡头垄断的方法有所不同。我们很难像分析其他市场那样绘制统一的成本曲线和利润最大化图，因为每个寡头市场都可能不同。因此，我们将更侧重于**博弈论 (Game Theory)** 和公司之间的**战略互动 (Strategic Interaction)**。

寡头垄断市场的主要特征如下：
1.  **少数公司**: 市场由少数几家公司主导，例如两家或五家。每家公司相对于市场规模都很大，其决策对整个市场有显著影响。
2.  **产品差异化程度低**: 为简化分析，我们通常假设产品是同质的或几乎相同。当然，也存在产品差异化的寡头市场，但其行为会更复杂。
3.  **进入壁垒**: 存在进入壁垒，但不一定是绝对的。例如，成为大学教授需要博士学位，这就是一种进入壁垒。虽然其他人可以通过努力获得学位后进入市场，但这需要时间和成本。其他例子包括行医执照等。

寡头垄断的例子非常普遍，包括：
- **软饮料行业**: 主要由可口可乐和百事可乐主导。
- **航空业**
- **手机服务**
- **卫星电视/广播**
- **汽车制造业**
- **原油行业** (注意：汽油零售市场更接近完全竞争，但原油生产是寡头垄断)

可以说，寡头垄断是第二常见的市场结构，仅次于垄断竞争。

## 战略互动与博弈论

寡头垄断与其他市场最关键的区别在于**战略互动**。在完全竞争市场，公司是价格接受者，无需关心其他公司的行为。在垄断市场，没有其他公司需要考虑。在垄断竞争市场，虽然公司会通过营销来吸引顾客，但由于每家公司规模相对较小，它们之间不存在真正的战略互动。

然而，在寡头垄断市场，每家公司的决策都会对市场产生重大影响，因此它们必须仔细考虑其他公司的行为。这导致了一个核心难题：寡头公司不确定自己面临的需求曲线是怎样的，因为这条曲线取决于竞争对手的决策。为了分析这种战略互动，我们使用**博弈论**。

博弈论是研究战略互动的学科。一个“博弈”包含以下要素：
- **参与者 (Players)**: 参与博弈的个人或公司。
- **策略 (Strategies)**: 参与者可以选择的行动方案。
- **收益 (Payoffs)**: 每种策略组合带来的结果，可以是好的也可以是坏的。

## 双头垄断：一个卖水的例子

我们通过一个简单的**双头垄断 (Duopoly)** 例子来说明。假设在一个小镇上，有两家公司销售水，并且为了简化，我们假设抽水和运水的**边际成本为零 (MC = 0)**。这意味着公司的总收入就等于其利润。

每天早上，两家公司都需要决定当天要带多少水到镇上销售。一旦到了镇上，就不能再增加供应量。水的市场价格取决于两家公司带来的总水量。如果一家公司预计另一家会带来大量的水，导致价格下降，它可能就会选择少带一些。反之，如果预计对方会少带水，它就可能选择多带一些以获取更高利润。双方都面临着这种不确定性。

我们来看一下这个小镇的需求表：

| 数量 (加仑) | 价格 ($) | 总收入/利润 ($) |
| :---------- | :------- | :---------------- |
| 0           | 120      | 0                 |
| 10          | 110      | 1100              |
| 20          | 100      | 2000              |
| 30          | 90       | 2700              |
| 40          | 80       | 3200              |
| 50          | 70       | 3500              |
| **60**      | **60**   | **3600**          |
| 70          | 50       | 3500              |
| 80          | 40       | 3200              |
| 90          | 30       | 2700              |
| 100         | 20       | 2000              |
| 110         | 10       | 1100              |
| 120         | 0        | 0                 |

### 两种极端情况：垄断与完全竞争

为了确定寡头垄断可能的结果，我们先分析两种极端情况：

1.  **垄断情况**: 如果市场上只有一家公司，它会选择利润最大化的产量。从上表可以看出，当产量为60加仑时，总利润最高，为3600美元，此时价格为60美元。这是垄断者会选择的结果。
2.  **完全竞争情况**: 在完全竞争市场中，价格等于边际成本。由于我们假设边际成本为零，价格也会被推到零。此时，市场总产量为120加仑，所有公司的利润都为零。

寡头垄断的结果将介于这两种极端之间，并且通常更接近垄断的结果。

### 勾结与卡特尔的困境

两家公司可能会意识到，如果它们能像一个垄断者那样行事，就能获得最大化的总利润。它们可以秘密达成协议，各自生产30加仑水，使总产量达到60加仑，价格为60美元，总利润为3600美元，每家公司分得1800美元。这种秘密协议被称为**勾结 (Collusion)**，形成一个**卡特尔 (Cartel)**。

然而，这种协议天生就不稳定。假设A公司和B公司达成了协议，都生产30加仑。A公司会这样想：“如果我确定B公司会遵守协议生产30加仑，那我该怎么做？”
- **遵守协议**: 我生产30加仑，总产量60，价格60，我的利润是 30 * 60 = 1800美元。
- **打破协议**: 我偷偷多生产10加仑，总共生产40加仑。那么市场总产量是70加仑，价格会降到50美元。我的利润将是 40 * 50 = 2000美元。而B公司的利润则降为 30 * 50 = 1500美元。

通过打破协议，A公司可以多赚200美元。同样，B公司也会面临完全相同的激励。因此，双方都有强烈的动机去打破协议。最终，很可能的结果是两家公司都生产40加仑，市场总产量达到80加仑，价格降至40美元。此时，每家公司的利润是 40 * 40 = 1600美元。

这个结果（各自获利1600美元）虽然不如合作（各自获利1800美元）好，但对每个独立的参与者来说，在对方遵守协议的情况下，背叛是更有利的选择。这就是个人利益与集体利益之间的冲突。

## 囚徒困境

这个现象可以用一个经典的博弈论模型——**囚徒困境 (Prisoner's Dilemma)** 来解释。

假设两名罪犯（A和B）因一项轻罪被捕，警方证据确凿，可以判他们各一年徒刑。但警方怀疑他们还犯下了一桩重罪，只是缺乏证据。于是，警方将他们分开关押，并向每个人提出以下条件：
- 如果你**坦白**并指证你的同伙，而他**保持沉默**，你将被无罪释放，而他将被判20年。
- 如果你们**都坦白**，你们将各被判8年。
- 如果你们**都保持沉默**，你们将因轻罪各被判1年。

我们可以用一个博弈矩阵来表示这个情景：

|            | **罪犯B：坦白** | **罪犯B：沉默** |
| :--------- | :-------------- | :-------------- |
| **罪犯A：坦白** | A: 8年, B: 8年  | A: 0年, B: 20年 |
| **罪犯A：沉默** | A: 20年, B: 0年 | A: 1年, B: 1年  |

从集体的角度看，最好的结果是两人都保持沉默，各判1年。但从个人角度看：
- **对罪犯A来说**:
    - 如果B坦白，A最好也坦白（8年 vs 20年）。
    - 如果B沉默，A最好还是坦白（0年 vs 1年）。
- **对罪犯B来说**，情况完全相同。

无论对方做什么，坦白都是更好的选择。因此，**坦白**是每个人的**占优策略 (Dominant Strategy)**。最终的可能结果是两人都坦白，各判8年。这个结果被称为**纳什均衡 (Nash Equilibrium)**。

**纳什均衡**是指在一个博弈中，每个参与者在给定其他参与者策略的情况下，都无法通过单方面改变自己的策略来获得更好的收益。换句话说，没有人会为自己的选择感到后悔。

### 合作的可能性

尽管存在囚徒困境，但在某些情况下，合作（或勾结）仍然是可能的：
1.  **重复博弈 (Repeated Game)**: 如果博弈会重复进行，参与者可以通过过去的经验学习，并意识到长期合作的好处。
2.  **惩罚机制 (Punishment)**: 如果存在有效的惩罚机制来惩罚背叛者，合作的可能性就会增加。
3.  **关系 (Relationship)**: 如果参与者之间有信任关系（如家人），他们更可能合作。
4.  **参与者数量少 (Small Number of Players)**: 参与者越少，达成并维持合作协议就越容易。

## 囚徒困境的现实应用

### 广告战
寡头公司（如可口可乐与百事可乐，或万宝路与骆驼香烟）的广告战就是囚徒困境的典型例子。对整个行业来说，如果所有公司都不打广告，它们可以节省大量费用，从而提高利润，因为广告并不能显著增加吸烟或喝可乐的总人数。然而，如果一家公司不打广告，而竞争对手打广告，它就会失去市场份额。因此，打广告成为了一个占优策略，导致所有公司都投入巨额广告费，最终利润反而低于都不打广告的情况。有趣的是，当美国政府禁止香烟公司在电视上做广告时，这些公司的利润反而上升了，因为政府替它们解决了这个囚徒困境。

### 军备竞赛
两个国家（如冷战时期的美国和苏联）面临着武装或裁军的选择。对双方来说，最好的结果是都裁军。但如果一方裁军而另一方不裁，裁军方将处于极度危险的境地。因此，武装自己成为了占优策略，导致了耗资巨大的军备竞赛。

### 负面竞选
在政治选举中，候选人面临着进行“干净”竞选还是“肮脏”竞选的选择。虽然选民普遍厌恶负面广告，但如果一个候选人进行负面攻击而对手不回应，攻击方往往能获得优势。因此，进行负面竞选也成了一个占优策略。

## 结论

1.  **寡头垄断中的勾结是困难的**: 尽管寡头公司有强烈的动机去勾结以获取垄断利润，但由于个人利益与集体利益的冲突（囚徒困境），这种合作关系天生就不稳定且难以维持。
2.  **寡头垄断的结果对社会有利**: 从社会整体福利的角度看，寡头公司难以成功勾结是一件好事。竞争（即使是不完全的）会使价格更低，产量更高，更接近于社会最优水平。
3.  **公司数量的影响**: 市场中的公司越多，达成并维持勾结协议就越困难。随着公司数量的增加，寡头市场的结果会越来越接近竞争市场。这也解释了为什么涉及大量参与者的阴谋论往往难以成立——人越多，保守秘密和维持合作就越困难。
