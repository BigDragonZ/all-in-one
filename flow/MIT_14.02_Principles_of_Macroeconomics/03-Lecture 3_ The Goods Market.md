这张图表展示了一项对经济学家进行的调查结果，他们被要求评估未来12个月内发生经济衰退的概率。**衰退（recession）** 基本上意味着总产出的下降。首先要注意的是，这不是什么好消息。根据这些专家的看法，美国在未来12个月左右进入衰退的可能性非常高。你可以看到，这个概率通常非常低，只有在衰退真正来临前才会急剧上升。虽然我们目前还未处于衰退之中，但人们普遍认为在不久的将来我们很有可能进入衰退。

那么，这些人是如何得出这些预测的呢？在某种程度上，无论是明确还是含蓄地，他们心中必然有一个关于均衡产出决定的模型。他们需要理解并观察某些迹象，这些迹象在通过模型分析后，表明产出将会下降。这就是我们今天开始要学习的内容，实际上，这也是我们整个课程将要做的。我们将贯穿整个课程，尝试寻找日益复杂或更丰富的模型来解释**总产出（aggregate output）**的决定因素。这基本上就是这门课的全部内容：我们将努力理解是什么驱动了均衡产出，以及我们是如何达到某个特定的产出水平的。这就是寻找均衡产出水平的意义。

---

我们将分三个阶段进行。在课程的第一部分，也就是第一次小测（quiz one）之前，我们将专注于**短期（short run）**，探讨在非常短的时间内（比如一年左右）产出是如何被决定的。然后，在课程的第二部分，我们将关注**中期（medium run）**。我们对中期的定义是，价格开始进行充分调整所需的时间。在此之前，大部分经济活动的变化体现在数量上，商品价格的变动很小，尽管资产价格可能会有很大波动。在课程的最后一部分，我们将研究产出在**长期（long run）**是如何决定的，这与短期的决定方式截然不同。

短期内的产出决定，我们通常称之为**商业周期（business cycle）**分析。我们在这里定义的短期或中期，就是商业周期的范畴。国家处于衰退、繁荣或扩张期，这些都是描述短期的术语。而长期均衡产出的决定，则关乎**增长（growth）**问题。当我们讨论为什么今天中国的增长速度超过美国时，这并非商业周期问题，而是关于产出的长期决定因素。在更高级的模型中（例如，如果你在攻读博士学位），这些概念会更加紧密地联系在一起。但在这门课程中，我们将它们视为非常不同类型的模型，因为用不同的模型来分别分析这些问题，比试图将所有东西整合到一个庞大的机器中要容易得多。

---

让我们从最简单的部分开始。在短期内，一个关键机制将反复出现在我们分析的所有模型和子模型中。这个机制就是：在极短期内，均衡产出主要是由我们所说的**需求（demand）**决定的。

1.  **需求决定产出**：需求的变化将改变生产。
2.  **产出改变收入**：当生产改变时，收入也会随之改变。正如你从国民账户中学到的，产出和收入是同一枚硬币的两面。更多的生产意味着有人（工人、资本所有者、政府等）会获得相应的收入。
3.  **收入再次影响需求**：当收入改变时，需求又会再次发生变化，如此循环往复。

这基本上是短期宏观经济学的精髓：理解**总需求（aggregate demand）**，因为它是主要驱动力，然后理解它在短期内是如何被放大的。这就是本讲的核心内容。当我提到短期宏观经济学时，我脑海中的结构就是这样：总需求决定一切。这就是为什么在短期内，人们非常关心消费者信心是高是低。如果消费者感到悲观，他们倾向于减少需求；如果他们非常乐观，则会增加需求。由于短期内产出由需求决定，商业周期（我们是否会陷入衰退）就取决于需求状况。因此，如果有人预测未来12个月内会发生衰退，他实际上是在预测未来12个月内需求将会下降。至于他们为什么这么预测，我们将在课程的后续步骤中学习，比如哪些因素可能会成为总需求的拖累，从而压低需求。

---

首先，让我介绍一下总需求的组成部分。

- **消费 (C)**：是总需求中最大的一部分，指消费者（家庭等）购买的商品和服务。
- **投资 (I)**：我们用 `I` 表示，是**非住宅投资**（如设备、工厂）和**住宅投资**（如房屋、公寓楼）的总和。这些也是商品和服务，只是它们是资本品。
- **政府支出 (G)**：我们用 `G` 表示，是联邦、州和地方政府购买的商品和服务。需要注意的是，这**不包括政府转移支付**。政府转移支付，比如你在新冠疫情期间可能收到的政府支票，它更像是一种负税收，会通过其他方式影响经济，而不是直接计入政府支出。
- **出口 (X)**：指外国人购买在美国生产的商品和服务。
- **进口 (IM)**：指美国消费者、公司和政府购买在国外生产的商品和服务。例如，你购买德国生产的东西是进口，而德国人购买美国生产的东西是出口。
- **存货投资 (Inventory Investment)**：指销售和生产之间的差额。这是一个波动性大但规模较小的部分，在本课程中，除非特别说明，我们将假设**存货投资为零**。

此外，在课程的这一部分，直到另行通知，我们都将假设**出口和进口也为零**。这虽然不现实，但分析一个不与世界其他地区互动的**封闭经济（closed economy）**会更容易。大约十讲之后，我们将开放经济，届时就需要讨论进口、出口、汇率等问题。但现在，让我们保持简单。

---

这张图表展示了2018年美国GDP的构成。虽然总量会变，但各部分的比例变化不大。你可以看到：

- **消费 (C)** 占了GDP的很大一部分，接近70%。这就是为什么人们如此关注消费者信心指数（比如密歇根大学发布的指数），因为它是均衡产出的主要驱动力之一。
- **投资 (I)** 的规模要小得多，但仍然很重要，特别是非住宅投资。
- **政府支出 (G)** 也是总需求的一个重要组成部分。
- 对于像美国这样的大国，**净出口 (X - IM)** 相对较小。对于一些小型经济体，出口占GDP的比重可能非常大。
- **存货投资** 的占比较小，这也是我们将其设为零的原因。虽然它波动很大，可能影响月度GDP数据，但在稍长的时间段内并不那么重要。

---

现在我们来构建模型。请务必跟上，因为从现在到第一次小测的所有内容都将建立在对这个简单模型的理解之上。如果你在这里错过了任何一步，接下来的几讲都会让你感到困惑。

我们将总需求用字母 **Z** 表示。在一个封闭经济且忽略存货投资的简化模型中，总需求由以下部分构成：
`Z ≡ C + I + G`

这里的 `≡` 是一个恒等式，表示这只是一个定义。我们的最终目标是确定美国经济的产出或生产。因此，当我们谈论总需求时，我们指的是对**国内生产的商品**的需求。

为了构建一个最简单的总需求模型，我们将做出以下假设：
- **投资 (I)** 和 **政府支出 (G)** 是外生给定的常量，我们用 `I` 和 `G` 上加一横来表示（尽管这里为了简化，我们直接写 `I` 和 `G`）。这意味着它们是由模型之外的因素决定的。
- **税收 (T)** 也是外生给定的。
- **消费 (C)** 是唯一内生变化的变量。我们将所有精力集中在消费上，因为它也是总需求中最重要的部分。

---

我们假设消费是**可支配收入 (disposable income, Yd)** 的一个增函数。这很自然：当你的可支配收入增加时，你会消费更多。

`C = C(Yd)` (其中 `+` 号表示正相关)

在现实中，消费函数要复杂得多，但我们从最基础的开始。为了进一步简化，我们假设消费是可支配收入的**线性函数**：
`C = C0 + C1 * Yd`

- **C0** 是**自主消费 (autonomous consumption)**。它代表了即使可支配收入为零，人们也必须进行的最低消费，或者捕捉了除当前收入外影响消费的所有其他因素（如财富、消费者信心等）。我们视其为一个外生参数。
- **C1** 是**边际消费倾向 (marginal propensity to consume)**。它表示每增加一美元的可支配收入，你会花多少钱用于消费。例如，如果你得到额外的一美元，花了60美分，那么你的 `C1` 就是0.6。`C1` 是一个介于0和1之间的数字。

---

**可支配收入 (Yd)** 的定义是总收入（Y）减去税收（T）。记住，从国民账户我们知道，总收入等于总产出（Y）。
`Yd ≡ Y - T`

将这个定义代入消费函数，我们得到：
`C = C0 + C1 * (Y - T)`

这张图描绘了消费函数。它是一条直线，截距是 `C0`，斜率是 `C1`。`C1` 介于0和1之间。

---

现在我们来确定均衡产出。我们已经有了总需求函数：
`Z = C + I + G`
`Z = [C0 + C1 * (Y - T)] + I + G`

这个方程描述了总需求与产出（或收入）之间的关系。但我们有两个未知数（Z 和 Y），只有一个方程，所以无法求解。我们需要第二个方程，也就是**均衡条件**。

均衡条件是：**产出 (Y) 等于总需求 (Z)**。
`Y = Z`

这**不是一个函数或恒等式**，它只在**均衡状态**下成立。它体现了短期宏观经济学的核心思想：产出将调整到与需求相等的水平。在非均衡状态下，`Y` 不等于 `Z`。

现在我们有了两个方程和两个未知数，可以求解了。将需求函数代入均衡条件：
`Y = C0 + C1 * (Y - T) + I + G`

通过代数运算，我们可以解出均衡产出 `Y`：
`Y * (1 - C1) = C0 + I + G - C1 * T`
`Y = [1 / (1 - C1)] * [C0 + I + G - C1 * T]`

---

这个方程中的两部分都非常重要：

1.  **[C0 + I + G - C1 * T]** 被称为**自主支出 (autonomous spending)**。这是不依赖于产出 `Y` 的那部分需求。
2.  **[1 / (1 - C1)]** 被称为**乘数 (multiplier)**。这是一个非常重要的宏观经济学概念，在微观经济学中不存在。

为什么叫乘数？因为 `C1` 介于0和1之间，所以 `(1 - C1)` 也介于0和1之间，这意味着乘数 `1 / (1 - C1)` **大于1**。它将自主支出的任何变化放大。

乘数的大小取决于 `C1`。当 `C1` 越大（越接近1），乘数就越大。这背后的逻辑是：当政府增加支出时，产出增加，从而导致收入增加。如果人们将大部分新增收入用于消费（即 `C1` 很大），这将进一步增加需求和产出，再次增加收入，形成一个正向循环。反之，如果人们非常谨慎，将大部分新增收入储蓄起来（`C1` 很小），那么这个循环就会很弱，乘数效应也就很小。

---

我们可以用一个图来直观地理解这个模型，这个图通常被称为**凯恩斯交叉图 (Keynesian Cross)**。

- **横轴**是产出/收入 (Y)。
- **纵轴**是需求 (Z)。
- **45度线**：代表了均衡条件 `Y = Z`。线上任意一点的横坐标都等于纵坐标。
- **总需求曲线 (ZZ)**：代表了需求函数 `Z = [C0 + I + G - C1*T] + C1*Y`。它的截距是自主支出，斜率是边际消费倾向 `C1`。因为 `C1` 小于1，所以ZZ曲线比45度线平缓。

**均衡点**是ZZ曲线与45度线的交点，该点的产出水平就是我们代数求解出的均衡产出 `Y`。在任何非均衡点，比如在交点左侧，需求大于产出；在交点右侧，产出大于需求。

---

让我们做一个思想实验：假设自主消费 `C0` 增加了10亿美元（比如因为消费者信心增强）。

1.  **初始冲击**：`C0` 增加10亿，导致总需求曲线ZZ向上平移10亿美元（从A点到B点）。
2.  **第一轮**：由于需求增加了10亿，产出也立即增加10亿以满足需求（从B点到C点）。
3.  **第二轮**：产出增加10亿意味着收入也增加10亿。消费者会将其中的 `C1` 比例用于消费。假设 `C1 = 0.5`，那么消费会额外增加5亿美元。这导致需求再次上升（从C点到D点）。
4.  **第三轮及以后**：这5亿美元的新增需求又会转化为5亿美元的新增产出和收入，然后又导致 `0.5 * 5亿 = 2.5亿` 的新增消费……

这个过程形成一个几何级数：`10亿 * (1 + C1 + C1² + C1³ + ...)`。这个级数的和等于 `10亿 * [1 / (1 - C1)]`。如果 `C1 = 0.5`，乘数就是2，最终均衡产出将增加 `10亿 * 2 = 20亿` 美元。这个过程在图上表现为从初始均衡点A沿着阶梯状路径最终到达新的均衡点A'。

---

还有一种完全等价的方法来推导均衡，即**投资等于储蓄 (Investment = Saving)**。这也是**IS曲线**名称的由来。

- **私人储蓄 (S)**：等于可支配收入减去消费。`S = Yd - C = Y - T - C`。
- **公共储蓄**：等于税收减去政府支出。`T - G`。如果 `T > G`，政府有预算盈余；如果 `T < G`，则有预算赤字（负储蓄）。
- **总储蓄**：等于私人储蓄加上公共储蓄。

在均衡状态下，**投资 (I) 必须等于总储蓄**。
`I = S + (T - G)`
`I = (Y - T - C) + (T - G)`
`I = Y - C - G`
`Y = C + I + G`

这与我们之前的均衡条件 `Y = Z` 完全相同。

---

这个“投资=储蓄”的框架可以很好地解释一个著名的宏观经济学概念：**节俭悖论 (paradox of saving)**。

我们从小被教育储蓄是美德。对个人而言，这通常是正确的。但在短期宏观经济中，如果整个社会突然决定增加储蓄，结果可能适得其反。

假设所有人都决定在给定的收入水平下储蓄更多。这意味着在任何收入水平下，消费都会减少（即自主消费 `C0` 下降）。
1.  **消费下降**：总需求曲线ZZ向下移动。
2.  **产出下降**：由于需求下降，均衡产出下降。
3.  **收入下降**：产出下降导致收入下降。
4.  **乘数效应**：收入的下降会进一步减少消费，导致产出和收入进一步下降，乘数效应在这里起了负面作用。

最终，整个经济陷入衰退。尽管每个人都想储蓄更多，但由于经济衰退导致收入下降，总储蓄水平可能并未增加，甚至可能下降。这就是节俭悖论：在短期内，个人追求储蓄的理性行为可能导致集体非理性的后果。

---

这个模型也揭示了政策制定者，尤其是在经济衰退时期，为什么总是试图安抚市场、提振信心。如果每个人都感到恐慌并削减开支，经济可能会仅仅因为悲观情绪而崩溃。

反之，在当前经济过热、通胀高企的时期，政策制定者（如美联储）可能希望给经济降温。他们通过发表鹰派言论（例如，“经济太热了，我们将不得不采取措施”），试图让消费者和市场稍微“悲观”一点，从而抑制需求。这就是为什么当美联储官员发表强硬讲话时，股市往往会下跌。

最后，我们可以用这个模型来理解**财政政策 (fiscal policy)**。
- **扩张性财政政策**：增加政府支出 `G` 或减少税收 `T`。这会提高总需求，通过乘数效应使产出增加更多。
- **紧缩性财政政策**：减少政府支出 `G` 或增加税收 `T`。这会降低总需求，通过乘数效应使产出减少更多。

因此，节俭悖论所展示的产出下降，与实施紧缩性财政政策的效果是类似的。两者都会导致总需求曲线下移，使经济收缩。
