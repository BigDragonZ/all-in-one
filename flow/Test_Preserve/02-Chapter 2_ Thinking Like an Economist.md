# Chapter 2: Thinking Like an Economist

In this video, we're going to talk about thinking like an economist. We will explore how economists approach problems and how we think about understanding human behavior.

The most basic thing we do is use the **scientific method**. We will think about something we want to explain and consider potential explanations for the behavior we observe. We will then test whether our hypothesis explains the behavior. If it doesn't, we will change our hypothesis. We use the scientific method very similarly to how it is used in a hard science.

One of the things we need to talk about is that we will be making **assumptions**. To understand human behavior, we need to start by making assumptions to simplify the problem. Economists are often criticized for making assumptions, but assumptions are made in all different disciplines. For example, suppose we were talking about a physics problem and trying to calculate the time it would take for this pen to hit the floor if I drop it. To calculate that using physics, we would make the simplifying assumption that it's dropped in a vacuum. This makes the problem much easier to figure out. If we don't make that assumption, we have to figure out the effect of wind resistance, which is a very complicated process. It depends on the shape of the pen, air temperature, humidity, and the exact way I drop it. By assuming it's in a vacuum, we can ignore all of that, and the problem becomes simpler. So, we are going to make simplifying assumptions.
