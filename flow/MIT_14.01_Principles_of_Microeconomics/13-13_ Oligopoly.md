# 寡头垄断与博弈论

## 导论：从理想到现实的市场结构

今天，我们终于要进入最现实的市场结构了。我们之前讨论过**完全竞争市场**，那是一个非常有用的极端例子，可以帮助我们思考经济效率。然后，我们转向了相对更现实一些的**垄断**市场，但现实中很少有市场只有一个参与者。真正的垄断在私营市场中是罕见的。

大多数市场的特征更符合**寡头垄断 (Oligopoly)**。这是一种由少数几家公司相互竞争的市场，同时存在进入壁垒，阻止了无限数量的公司进入。我们可以将这类市场看作存在一定的进入壁垒，所以公司不能像在 IBM/Dell 的例子中那样无成本地自由进出。但这些壁垒又足够低，以至于有几家公司已经进入，而不仅仅是一家。所以，它不是自然垄断，不是只有一家公司能存在的市场。市场中有多家公司，但它们只需要彼此竞争，而不用与整个世界竞争。

一个典型的寡头垄断例子是汽车行业。汽车制造商之间显然在竞争，只要你看任何体育赛事，看看他们投入了多少广告，就知道他们一直在相互较劲。但世界上绝大多数的汽车是由不到十家汽车制造商生产的。因此，那种认为有成千上万的卖家销售完全相同商品的完全竞争市场的观念，在购买汽车这个情境下显然是不成立的。这就是我们接下来几节课要重点关注的模型。

## 寡头垄断中的合作与非合作行为

在寡头垄断市场中，这些有限的竞争者可以有两种行为方式：**合作**或**非合作**。

合作意味着它们可以组成所谓的**卡特尔 (Cartel)**。当寡头市场中的公司合作起来共同做决策时，这就叫卡特尔。其中最著名的例子是 **OPEC（石油输出国组织）**。这是一个由控制着世界约三分之二石油储量的国家组成的组织，由沙特阿拉伯主导。这个由大约十几个国家组成的卡特尔，通过合作行为，实际上将自己变成了一个垄断者。OPEC 的行为就像它垄断了石油一样。当然，这在过去更为真实，现在随着其他非 OPEC 国家开始生产更多石油，情况变得更加困难。但很长一段时间里，它们是石油的合作供给方，行为与垄断者无异：保持高价格，将产量维持在无效率的低水平，并像垄断者一样赚取了大量利润。

然而，这对生产者来说是很好的结果，但正如我们下次会讨论的，这种合作关系实际上很难维持。事实证明，维持卡特尔的团结非常困难。因此，寡头垄断市场通常以**非合作**的方式运作，参与者之间相互竞争，而非合作。在这种情况下，它们的利润可能会被压低到远低于垄断水平，甚至可能一直降到完全竞争的水平。

你可以把市场看作一个谱系：一端是完全竞争，另一端是垄断。寡头垄断介于两者之间。一个合作的寡头市场，比如卡特尔，其结果会接近垄断。而非合作的市场则处于两者之间的某个位置。我们今天就要建立模型，来探究它们最终会落在哪个位置。

## 博弈论入门：纳什均衡

要思考这个问题，我们必须借助一个在过去 40 年里成为经济学主流工具的理论：**博弈论 (Game Theory)**。

我们将把寡头垄断公司看作是在进行一场**博弈**。和任何博弈一样，你需要知道两件事：一是**策略**是什么，二是博弈何时结束，也就是**均衡**是什么。博弈论的关键在于定义均衡，从而推导出参与者将采取的策略。然而，在非合作的寡头垄断博弈中，均衡并没有一个像游戏规则那样明确的定义。你必须提出不同的均衡概念。

最常用的均衡概念叫做**纳什均衡 (Nash Equilibrium)**，以著名数学家约翰·纳什 (John Nash) 的名字命名。经济学家们已经把他当作自己人，尽管他其实是位数学家，但他还是获得了诺贝尔经济学奖。你们可能都通过电影和书籍《美丽心灵》了解他，他基本上是博弈论之父。

那么，什么是纳什均衡？纳什均衡的定义是：**在给定其他参与者策略的情况下，没有任何一个参与者愿意单方面改变自己的策略。** 换句话说，在纳什均衡点上，考虑到其他所有人的决定，每个参与者都对自己当前的决定感到满意。如果我有一个策略，并且在其他参与者使用他们策略的前提下，我仍然对自己的策略感到满意，那么这就构成了一个均衡。

## 经典案例：囚徒困境

这是一个非常抽象的概念，让我们用一个经典的例子来说明：**囚徒困境 (Prisoner's Dilemma)**。

这个场景常见于老式警匪片：警察逮捕了两个嫌疑人，把他们分开关在不同的房间里分别审讯。警察告诉每个人以下规则：
- 目前的证据足以让两人各判一年。
- 如果你指证你的同伙，而他保持沉默，那么你将被无罪释放，而他将被判五年。
- 如果你的同伙指证你，而你保持沉默，那么他将被无罪释放，你将被判五年。
- 如果你们互相指证，那么两人将各判两年。

为了分析这个决策，我们通常会构建一个**支付矩阵 (Payoff Matrix)**。

| | **囚犯 B：沉默** | **囚犯 B：招供** |
| :--- | :--- | :--- |
| **囚犯 A：沉默** | A: 1年, B: 1年 | A: 5年, B: 0年 |
| **囚犯 A：招供** | A: 0年, B: 5年 | A: 2年, B: 2年 |

现在我们来分析，每个囚犯的最佳策略是什么？在纳什均衡的概念下，我们寻找**占优策略 (Dominant Strategy)**，即无论对方做什么，对我来说都是最优的策略。

- **从囚犯 A 的角度看**：
    - 如果 B 保持沉默，A 招供（0年）比沉默（1年）更好。
    - 如果 B 招供，A 招供（2年）也比沉默（5年）更好。
    - 因此，无论 B 怎么做，A 的占优策略都是**招供**。

- **从囚犯 B 的角度看**（由于对称性）：
    - 无论 A 怎么做，B 的占优策略同样是**招供**。

最终，非合作的均衡结果是两人都选择招供，各判两年。这就是一个“逐底竞争”的例子。这个非合作均衡对两个参与者来说，都比他们本可以达成的合作均衡（都保持沉默，各判一年）要糟糕得多。

这是纳什的一个惊人洞见。在纳什之前，我们普遍认为竞争总是好的。但纳什第一个指出，竞争也可能导致坏的结果，合作有时反而更好。就像《美丽心灵》中那个在酒吧里追女孩的场景，他描述了竞争如何导致了最差的结局。

## 经济学应用

### 广告战：可口可乐 vs. 百事可乐

这个模型可以应用于经济学中的广告决策。假设可口可乐和百事可乐需要决定是否投放广告。

- **合作策略**：双方可以达成协议，都不打广告，共同瓜分市场并节省巨额广告费。
- **非合作策略**：让我们构建一个支付矩阵（利润单位：十亿美元）：

| | **百事：不广告** | **百事：广告** |
| :--- | :--- | :--- |
| **可口可乐：不广告** | 可口可乐: 8, 百事: 8 | 可口可乐: -2, 百事: 13 |
| **可口可乐：广告** | 可口可乐: 13, 百事: -2 | 可口可乐: 3, 百事: 3 |

分析这个矩阵：
- **从可口可乐的角度看**：
    - 如果百事不广告，可口可乐广告（13）比不广告（8）更好。
    - 如果百事广告，可口可乐广告（3）比不广告（-2）更好。
    - 可口可乐的占优策略是**广告**。
- **从百事的角度看**：
    - 百事的占优策略同样是**广告**。

因此，均衡结果是双方都选择广告，各自只赚 30 亿，而如果它们能合作不打广告，本可以各赚 80 亿。一个真实的例子是烈酒行业。在我小时候，电视上从来看不到烈酒广告，这不是因为法律禁止，而是因为整个行业合作达成了不打广告的协议，维持了一个合作均衡。后来这个协议被打破，现在所有品牌都在打广告，它们的整体利润可能还不如从前。

### 重复博弈：合作的可能性

囚徒困境的悲观结果并非无法改变。博弈论中一个令人兴奋的进展是引入了**重复博弈 (Repeated Game)** 的概念。

想象一下，可口可乐对百事可乐说：“我保证不打广告，只要你也不打。但只要你打一次广告，我就会永远打广告。”

现在，百事可乐在第一期做决策时，它有两个选择：
1.  **背叛**：在第一期打广告，可以赚到 130 亿的短期高额利润。但从第二期开始，可口可乐会永远打广告，导致百事在未来的每一期都只能赚 30 亿。
2.  **合作**：不打广告，与可口可乐一起维持合作均衡，那么它在未来的每一期都能赚到 80 亿。

很明显，长期来看，“永远赚 80 亿”远比“赚一次 130 亿，然后永远赚 30 亿”要好得多。通过将一次性博弈变成重复博弈，可口可乐实际上解决（或强制实现）了囚徒困境中的合作均衡。

然而，这个解决方案非常脆弱。它只在博弈**无限期**进行时才有效。一旦博弈有一个已知的终点（比如，政府宣布十年后将禁止销售汽水），整个合作就会崩溃。在第九年，百事会想：“这是最后一年了，我最好打广告赚一把，因为明年就没有惩罚了。” 可口可乐也知道这一点，所以它也会在第九年打广告。这个逻辑会向前倒推：既然第九年注定要竞争，那第八年也没有合作的必要了……最终，从第一年开始，双方就都会选择打广告。

这就是博弈论的精妙之处，它为我们提供了思考这些复杂决策的工具。

## 古诺模型：寡头垄断的数学化

现在，让我们将这些博弈论的概念应用到一个更数学化的模型中：**古诺模型 (Cournot Model)**，用于分析非合作的寡头垄断。

假设有两家航空公司，**美联航 (United)** 和**美国航空 (American)**，它们是唯一运营波士顿到芝加哥航线的公司。它们需要决定每天运营多少航班（即产量 Q）。

**古诺均衡**是一种纳什均衡，其中每家公司都选择了一个利润最大化的产量，前提是给定另一家公司的产量。

### 求解古诺均衡的步骤

1.  为每家公司求解其**剩余需求函数 (Residual Demand Function)**，即在另一家公司满足部分市场需求后，留给它的需求。
2.  基于剩余需求，推导出公司的**边际收益函数 (Marginal Revenue Function)**，这个函数将依赖于竞争对手的产量。
3.  令 **MR = MC**，求解出公司的最优产量，这个产量是其竞争对手产量的函数。这个函数被称为**反应曲线 (Reaction Curve)** 或**最佳反应曲线 (Best Response Curve)**。
4.  为另一家公司重复以上步骤，得到它的反应曲线。
5.  求解由两条反应曲线构成的方程组，得到两家公司各自的均衡产量。

### 实例分析：航空公司

假设市场需求函数为 `P = 339 - Q` (其中 `Q = Q_A + Q_U`)，两家公司的边际成本 (MC) 均为 147 美元。

1.  **基准：垄断情况**
    如果只有美国航空一家，其收益为 `R = P * Q_A = (339 - Q_A) * Q_A = 339Q_A - Q_A^2`。
    其边际收益为 `MR_A = 339 - 2Q_A`。
    令 `MR = MC`，即 `339 - 2Q_A = 147`，解得垄断产量 `Q_A = 96`。
    价格 `P = 339 - 96 = 243` 美元。

2.  **古诺双寡头模型**
    现在考虑美联航的存在。美国航空面对的剩余需求为 `P = 339 - (Q_A + Q_U)`。
    其总收益为 `R_A = P * Q_A = (339 - Q_A - Q_U) * Q_A = 339Q_A - Q_A^2 - Q_A * Q_U`。
    对 `Q_A` 求导得到边际收益：`MR_A = 339 - 2Q_A - Q_U`。
    令 `MR_A = MC`，即 `339 - 2Q_A - Q_U = 147`。
    整理得到美国航空的**反应曲线**：`Q_A = 96 - 0.5 * Q_U`。

3.  **求解均衡**
    由于两家公司是对称的（成本相同），美联航的反应曲线为：`Q_U = 96 - 0.5 * Q_A`。
    我们现在有两个方程和两个未知数。将第二个方程代入第一个方程：
    `Q_A = 96 - 0.5 * (96 - 0.5 * Q_A)`
    `Q_A = 96 - 48 + 0.25 * Q_A`
    `0.75 * Q_A = 48`
    解得 `Q_A = 64`。
    由于对称性，`Q_U = 64`。

    这是古诺均衡产量。此时，市场总产量 `Q = Q_A + Q_U = 64 + 64 = 128`。
    均衡价格为 `P = 339 - 128 = 211` 美元。

在这个均衡点上，两家公司都对自己的产量（64）感到满意，因为在对方产量为 64 的情况下，64 是自己的最佳选择。没有一家公司有动机单方面改变产量。

当然，现实世界可能更复杂，比如公司成本不同（非对称），或者反应曲线非线性，可能导致多个均衡甚至没有均衡。但这个简单的古诺模型为我们理解寡头竞争提供了一个强大的基础框架。
