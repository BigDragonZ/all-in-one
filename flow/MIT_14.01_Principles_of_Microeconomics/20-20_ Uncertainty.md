### 在不确定性下做决策

可以思考一下你为这门课的期末考试复习的决定。在我们之前的模型中，你可以根据课程的不同单元来优化你的学习。你会知道期末考试中每个单元的占比，并据此优化你的学习安排。但实际上你并不知道这些，你面临着关于期末考试会涵盖哪些内容以及各部分比例的不确定性。虽然你知道整个课程都是考察范围，但在分配时间时，你并不确定该如何在不同主题上分配时间。那么，你该如何做出这个决定呢？

我们需要运用我们的工具来思考在不确定性下的这类决策情景。这不仅仅是关于考试。比如，我今天决定是否带伞。如果我带了伞，我可能会弄丢它，但我又不想被淋湿。我必须考虑是否会下雨，这完全取决于我对下雨的确定性有多大。还有关于体育赛事下注的决定，这也是在不确定性下做决策。

当你进入充满焦虑的成年生活后，你会面临更多类似的问题，比如是选择7年期抵押贷款还是30年期抵押贷款，是否购买健康保险，把孩子送到哪所学校。所有这些事情都包含了巨大的不确定性。

### 预期效用理论

我们尚未建立处理这些问题的工具。但经济学有一个非常有用的工具，可以用来思考这类情况，这个工具就像我们本学期学过的其他工具一样，一旦你理解了它，就会发现它非常简单，并且在解释世界方面具有巨大的力量。这个工具就是 **预期效用理论 (Expected Utility Theory)**，这也是我们今天将要学习的重点。

现在，我来问你一个问题，请凭直觉快速回答，不要想太多。我想和你打个赌（当然是假设的）：我抛一枚硬币。

> 正面朝上，你得到125美元。
> 反面朝上，你给我100美元。

有多少人愿意接受这个赌注？（大约40%的人举手）你们这届学生比平常更大胆一些，通常我得到的比例是20%到25%。

那些举手的人可能很快在脑子里做了一个我们称之为**期望值 (Expected Value)** 的计算。任何赌博的期望值计算公式是：
`期望值 = (获胜概率 × 获胜收益) + (失败概率 × 失败损失)`

只要我没有作弊，用的是一枚公平的硬币，那么计算就是：
`0.5 * $125 + 0.5 * (-$100) = $12.50`

这个赌注的期望值是12.50美元。这是一个正的期望值，在经济学中我们称之为“**优于公平的赌局 (more than fair bet)**”。一个公平的赌局（fair bet）是指期望值为零的赌局。所以，一个期望值为正的赌局就是优于公平的。

然而，你们中许多人没有接受这个赌注，这并不意味着你们错了，只是说明期望值不是思考这个问题的正确方式。正确的思考方式是，你关心的不是金钱，而是**效用 (utility)**。金钱本身没有意义，作为消费者，你关心的是你的效用。因此，我们不应该考虑期望值，而应该考虑**预期效用 (Expected Utility)**。

预期效用的公式与期望值类似，但有一个变化：
`预期效用 = (获胜概率 × 获胜时的效用) + (失败概率 × 失败时的效用)`

这与期望值有所不同，因为效用并非金钱的线性函数，而是金钱的**凹函数 (concave function)**。因此，效用函数表现出**边际效用递减 (diminishing marginal utility)**。这意味着下一个一美元带给你的价值要小于前一个一美元。就像下一片披萨带给你的满足感不如前一片一样。因此，失去一美元让你感到的悲伤，要比赢得一美元让你感到的快乐更多。

### 风险规避

让我们用一个我们之前在消费者理论中用过的典型效用函数来举例：`U = √C`，其中 `C` 是消费。我们假设你花掉所有收入。假设你最初的消费 `C₀` 是100美元，那么你的初始效用 `U₀` 就是 `√100 = 10`。

现在我向你提供那个赌注。这个赌注的预期效用是多少？
- 如果你赢了，你的消费变成 `100 + 125 = 225` 美元。
- 如果你输了，你的消费变成 `100 - 100 = 0` 美元。

预期效用是：
`预期效用 = (0.5 * U(225)) + (0.5 * U(0)) = (0.5 * √225) + (0.5 * √0) = (0.5 * 15) + 0 = 7.5`

这个赌注的预期效用是7.5，低于你10的初始效用。所以，你不应该接受这个赌注。这也就是为什么你们中许多人没有接受的原因。你不应该接受这个赌注，因为你是我们所说的**风险规避者 (risk averse)**。

风险对我们来说具有内在的负价值。一个确定的一美元比一个不确定的一美元对我们更有价值。就像今天的一美元比明天的一美元更有价值一样。

要理解这一点，最好的方式是通过图形。在图20-1中，x轴是你的财富（或消费），y轴是你的效用。你可以看到这是一个凹形图，它表现出边际效用递减。财富的每一美元都会增加效用，但增加的幅度越来越小。

- 你从100美元的财富开始，对应10的效用（点A）。
- 赌博提供了两种可能的结果：财富为225美元（效用为15，点B），或财富为0（效用为0）。
- 这个赌博的预期效用是这两个效用点的中点，即7.5。这低于你保持100美元财富所获得的10的效用。
- 这是因为从100美元降到0让你感到的悲伤，远远超过从100美元增加到225美元让你感到的快乐。这完全是由于收入的边际效用递减。

利用这个图，我们可以问：你愿意付多少钱来**避免**这个赌局？
这个赌局带给你的预期效用是7.5。在图上，7.5的效用水平对应的财富是 `7.5² = 56.25` 美元。这意味着，你宁愿直接拥有56.25美元的确定财富，也不愿参与这个期望值为112.5美元（`0.5*225 + 0.5*0`）的赌局。因此，你愿意支付 `100 - 56.25 = 43.75` 美元来避免这个赌局。这就是你的**风险溢价 (risk premium)**。

另一个问题是：为了让你接受这个赌局（输100美元），赢的金额X需要多大？
你需要让预期效用至少等于你现有的效用10。
`预期效用 = (0.5 * U(100+X)) + (0.5 * U(0)) = 10`
`(0.5 * √(100+X)) = 10`
`√(100+X) = 20`
`100 + X = 400`
`X = 300`
我需要提供“输100，赢300”的赌局，你才会接受。这再次说明了风险规避的原则。

### 风险中性与风险偏好

现在，让我们改变一下效用函数。
假设你的效用函数是线性的：`U = 0.1 * C`。你的初始财富仍然是100，初始效用是 `0.1 * 100 = 10`。
现在，这个赌注的预期效用是：
`预期效用 = (0.5 * U(225)) + (0.5 * U(0)) = (0.5 * 0.1 * 225) + (0.5 * 0.1 * 0) = 11.25`
这个值大于10，所以你会接受这个赌注。这是因为线性效用函数没有边际效用递减。这种情况我们称之为**风险中性 (risk neutral)**。风险中性的人只关心期望值，他们总是会接受优于公平的赌局。

再进一步，想象一个凸函数：`U = C² / 1000`。初始效用仍然是 `100² / 1000 = 10`。
这个赌注的预期效用是：
`预期效用 = (0.5 * U(225)) + (0.5 * U(0)) = (0.5 * 225² / 1000) + 0 ≈ 25.3`
这个值远大于10，所以你肯定会接受这个赌注。为什么？因为现在你是**风险偏好者 (risk loving)**。这种效用函数具有**边际效用递增**的特性，下一片披萨比上一片让你更快乐。在这种情况下，你甚至会接受一个不公平的赌局。

### 赌注大小与行为经济学

现在考虑一个新的赌注：赢12.5美元，输10美元。你接受吗？
这次，更多的人应该接受。让我们用最初的风险规避效用函数 `U = √C` 来计算：
`预期效用 = (0.5 * √112.5) + (0.5 * √90) ≈ (0.5 * 10.6) + (0.5 * 9.5) = 10.05`
这个值大于10，所以你应该接受这个赌注。发生了什么变化？你仍然是风险规避者，但现在你愿意接受这个赌注了。
原因是，当赌注相对于你的初始财富变得越来越小时，你的行为会越来越接近风险中性。因为在效用曲线上，一个很小的局部可以近似看作是线性的。

然而，为什么即使是这样的小赌注，仍然有很多人不愿接受？这引出了一个行为经济学的概念：**损失规避 (loss aversion)**。人类不仅是风险规半岛体，还具有一种非理性的行为偏见，即失去本身会让我们感到悲伤。拿走我们拥有的东西，比给予我们新东西更能影响我们的情绪。
一个经典的实验是：随机将人分成两组，问第一组“你愿意花多少钱买这个马克杯？”；给第二组每人一个马克杯，然后问“你愿意以多少钱卖掉你的马克杯？”。结果发现，人们平均愿意花3美元购买，但需要7美元才愿意卖出。这在标准经济模型中没有意义，但它揭示了人们的起始点会影响他们的决策。

### 应用一：保险

风险规避理论的一个重要应用是**保险**。在美国，个人每年在保险产品上的花费高达1.5万亿美元，几乎占GDP的10%。为什么？因为人们是风险规避的。

让我们举个例子：一个25岁的健康男性，年收入4万美元。他每年有1%的概率被车撞，如果发生，将产生3万美元的医疗费用。
- **没有保险**：他的预期效用是 `(0.99 * √40000) + (0.01 * √10000) = (0.99 * 200) + (0.01 * 100) = 198 + 1 = 199`。
- **有保险**：假设保险费为X。无论是否发生事故，他的财富都是 `40000 - X`。他的确定效用是 `√(40000 - X)`。

他愿意支付的最高保费是多少？我们让两种情况的效用相等：
`√(40000 - X) = 199`
`40000 - X = 199² = 39601`
`X = 399`
他愿意支付高达399美元的保费。而这个风险的期望成本只有 `0.01 * 30000 = 300` 美元。他愿意多付的99美元就是**风险溢价**。这解释了为什么保险业是一个巨大的产业。

风险溢价的大小取决于风险相对于你收入的大小。风险越大，或者你的收入越低，风险溢价就越高，你就越愿意购买保险。这就是为什么对于昂贵的汽车音响，低收入者可能比我更愿意购买延长保修（一种保险）。

### 应用二：彩票

另一个应用是**彩票**。彩票完全是个骗局，它的期望回报率大约是50%，是一个远差于公平的赌局。然而，彩票却非常受欢迎。为什么人们会玩彩票？有几种理论：

1.  **人们是风险偏好的**：这个理论是错误的，因为如果人们是风险偏好的，他们就不会购买保险。
2.  **弗里德曼-萨维奇偏好 (Friedman-Savage Preferences)**：该理论认为，人们在小赌注上是风险规避的，但在可能带来巨大财富的大赌注上是风险偏好的。这可以解释为什么有人会买超级百万（Mega Millions）彩票。然而，这个理论无法解释为什么绝大多数彩票消费是小额的刮刮乐，人们在这些小赌注上也表现出风险偏好的行为。
3.  **娱乐价值**：人们从等待开奖的刺激感中获得效用。花几美元买一张彩票，可以获得一段时间的娱乐和梦想。从这个角度看，彩票是一种“自愿税”，政府通过提供娱乐来筹集资金。
4.  **无知**：俗话说，“彩票是向愚蠢的人征收的税”。人们可能根本不理解这是一个多么糟糕的交易。

我们不确定后两种理论哪个是正确的，但它们对政府政策有截然不同的启示。如果彩票是娱乐，那么政府支持彩票是增进社会福利的。但如果是因为无知，那么政府应该阻止人们浪费他们的钱。这是一个重要的问题，尤其是在许多低收入社区，彩票支出占了他们收入的很大一部分。
