## 一、核心定义

### 1.1 稀缺性（Scarcity）
社会拥有无限的欲望，但资源是有限的。因此，社会无法生产所有人们想要消费的商品和服务。

### 1.2 经济学十大原理

| 类别 | 原理 | 核心公式 |
|------|------|----------|
| 人们如何决策 | 1. 人们面临权衡取舍 | — |
| | 2. 某种东西的成本是为了得到它所放弃的东西（机会成本） | $OC = \text{次优选择的价值}$ |
| | 3. 理性人思考边际 | $MB = MC$ 时达到最优 |
| | 4. 人们会对激励做出反应 | — |
| 人们如何相互交易 | 5. 贸易能使每个人状况更好 | — |
| | 6. 市场通常是组织经济活动的一种好方法 | — |
| | 7. 政府有时可以改善市场结果 | — |
| 整体经济如何运行 | 8. 一国的生活水平取决于它生产物品与服务的能力 | — |
| | 9. 当政府发行了过多货币时，物价上升 | $MV = PY$ |
| | 10. 社会面临通货膨胀与失业之间的短期权衡取舍 | 菲利普斯曲线 |

### 1.3 边际分析的数学形式化

**边际收益 (Marginal Benefit, $MB$)**：
$$MB = \frac{d(TR)}{dx}$$

**边际成本 (Marginal Cost, $MC$)**：
$$MC = \frac{d(TC)}{dx}$$

**最优决策规则**：
$$\text{若 } MB > MC, \text{ 增加活动} \\
\text{若 } MB < MC, \text{ 减少活动} \\
\text{最优状态：} MB = MC$$

### 1.4 实证分析与规范分析

| 维度 | 实证分析 (Positive) | 规范分析 (Normative) |
|------|---------------------|----------------------|
| 问题 | "是什么" (what is) | "应该是什么" (what should be) |
| 性质 | 描述性 | 规定性 |
| 可检验性 | 可通过数据证实/证伪 | 包含价值观，无法仅通过数据判断 |
| 例子 | "最低工资会导致失业" | "政府应该提高最低工资" |

## 二、数理/逻辑推导

### 2.1 生产可能性边界（PPF）的凹性与机会成本递增

设经济体生产两种商品 $X$（汽车）和 $Y$（电脑），生产函数为 $Y = f(X)$。

**机会成本定义**：
$$OC_X = \left| \frac{dY}{dX} \right| = |f'(X)|$$

**递增规律的数学表达**：
PPF 凹向原点意味着随着 $X$ 增加，曲线越来越陡峭：
$$\frac{d(OC_X)}{dX} > 0$$

**经济学解释**：资源具有专业化特征。最初转移的是最不适合生产电脑、最适合生产汽车的资源（成本低）；随着汽车产量增加，必须动用极度适合生产电脑的资源，导致放弃的电脑数量剧增。

### 2.2 循环流量图（Circular Flow Diagram）

| 市场 | 家庭行为 | 企业行为 |
|------|----------|----------|
| 产品与服务市场 | **买方**：支付货币（支出），换取产出 | **卖方**：提供产出，换取货币收入 |
| 生产要素市场 | **卖方**：提供土地、劳动和资本，换取收入 | **买方**：购买投入品，支付报酬 |

**对称性体现**：
- 在一个市场中是"卖方"的主体，在另一个市场必然是"买方"
- 货币与实物的反向循环保证了经济体内部循环的连续性

## 三、学术批判

### 3.1 科学方法在经济学中的根本局限

1. **无法实现真正的可控实验**：经济学研究的对象是具有主观能动性的人类行为。研究人员无法在宏观经济中实施"其他条件不变" (Ceteris Paribus) 的实验。

2. **主体反应性 (Reactivity)**：一旦政策制定者根据某种模型制定规则，微观主体会根据预期调整行为，从而使原有的模型关联失效。

3. **道德与现实的约束**：经济学家不能通过剥夺一群人的教育机会来实验"教育对收入的影响"，这使得实证分析高度依赖历史数据而非主动实验。

### 3.2 假设不现实是否意味着模型无效？

**Milton Friedman 的 'as if' 方法论**：
- 模型是否有效，不取决于其假设是否"真实"，而取决于其**预测和解释力**
- 我们假设"人是理性的"并不意味着现实中每个人都是天才数学家
- 只要人们的行为**表现得好像 (as if)** 他们在进行边际成本与收益的权衡，那么这个假设就是成立的

### 3.3 贸易原理（Principle 5）的失效条件

虽然贸易是正和博弈，但"使每个人更好"的结论在以下边界条件下可能失效：

1. **分配效应导致的"内部受损者"**：贸易虽然能增加总剩余，但会导致蛋糕分配不均
2. **市场失灵 (Market Failures)**：如果生产过程存在负外部性，而贸易导致生产规模扩大，增加的外部性成本可能抵消贸易所得
3. **非对称信息与市场势力**：如果一方拥有垄断势力，贸易可能演变为剩余的榨取

### 3.4 激励原理（Principle 4）的桥梁作用

原理 4 是微观决策与宏观效果之间的**因果粘合剂**：
- **微观层面**：理性个体通过比较 $MB$ 和 $MC$ 做出选择
- **宏观层面**：任何宏观政策（税收、补贴、价格管制）实质上都在改变这些边际变量
- **预测不可预见的后果**：忽略激励会导致政策产生"意外效果"
