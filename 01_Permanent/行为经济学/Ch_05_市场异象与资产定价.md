## 一、核心定义

### 1.1 股票溢价之谜（Equity Premium Puzzle）

**Mehra-Prescott模型**：
- 历史风险溢价约6.18%
- 模型预测最高溢价仅0.35%
- 匹配6.18%需风险厌恶系数γ=30

**含义**：
- 传统理论无法解释股票高溢价
- 投资者极端风险厌恶不现实

### 1.2 过度波动之谜（Excess Volatility）

**Shiller方差不等式**：
$$Var(P_t) \leq Var(P_t^*)$$

其中 $P_t^*$ 为事后理性价格（未来股利贴现值）。

**实证**：
- 真实股价波动远大于理性价格
- 股价波动非纯由基本面驱动

### 1.3 横截面异象

**规模效应（Size Effect）**：
- 小盘股超额收益

**账面市值比效应（B/M Effect）**：
- 高B/M股票（价值股）超额收益

**动量效应（Momentum）**：
- 赢家组合持续跑赢

**反转效应（Reversal）**：
- 输家组合长期反转

## 二、数理/逻辑推导

### 2.1 Mehra-Prescott模型

**相对效用函数**：
$$U(C) = \frac{C^{1-\gamma}}{1-\gamma}$$

**风险溢价**：
$$E(R_e) - R_f = \gamma \cdot \sigma_c^2$$

**矛盾**：
- γ=30意味着极端风险厌恶
- 50%得5万、50%得10万 → 宁愿要确定的5.12万

### 2.2 三大行为模型

**BSV模型**：
- 保守主义 → 反应不足 → 动量
- 代表性启发 → 反应过度 → 反转

**DHS模型**：
- 过度自信 → 短期反应过度 → 动量
- 自我归因 → 长期回归 → 反转

**JH模型**：
- 处置效应 → 盈利信息融入缓慢 → 动量

### 2.3 规模效应与价值效应

**代表性启发**：
- "好公司即好投资"
- 高估成长股可预测性
- 忽视小公司和价值股

**结果**：
- 成长股高估
- 价值股低估
- 业绩回归均值时产生超额收益

## 三、学术批判

### 3.1 风险因子解释

**Fama-French三因子**：
- 市场因子（MKT）
- 规模因子（SMB）
- 价值因子（HML）

**五因子扩展**：
- 盈利因子（RMW）
- 投资因子（CMA）

**争议**：
- 因子是风险补偿还是定价错误？
- 行为解释 vs. 理性解释

### 3.2 数据挖掘问题

**生存偏差**：
- 仅分析存活公司
- 忽略退市公司

**前视偏差**：
- 使用未来信息
- 不可复制

**多重检验**：
- 大量检验偶然发现显著性
- 需要调整显著性水平

### 3.3 策略衰减

**公开后失效**：
- 信息传播
- 纠正性套利
- Alpha消失

**动态变化**：
- 市场效率提升
- 竞争加剧
- 策略生命周期缩短
