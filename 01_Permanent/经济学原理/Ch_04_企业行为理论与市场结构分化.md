## 一、核心定义

### 1.1 利润最大化原则

**总利润**：
$$\pi = TR - TC$$

**边际收益（MR）**：
$$MR = \frac{dTR}{dQ}$$

**边际成本（MC）**：
$$MC = \frac{dTC}{dQ}$$

**利润最大化条件**：
$$MR = MC$$

### 1.2 成本曲线

**短期成本**：
- 固定成本（FC）：不随产量变动
- 可变成本（VC）：随产量变动
- 总成本：$TC = FC + VC$

**平均成本**：
- $ATC = \frac{TC}{Q}$
- $AVC = \frac{VC}{Q}$
- $AFC = \frac{FC}{Q}$

**边际产量递减**：
$$MP_L = \frac{\Delta Q}{\Delta L}, \quad \frac{dMP_L}{dL} < 0$$

### 1.3 市场结构对比

| 维度 | 完全竞争 | 垄断竞争 | 寡头 | 垄断 |
|------|----------|----------|------|------|
| **企业数量** | 很多 | 很多 | 几个 | 一个 |
| **产品差异** | 同质 | 有差异 | 同质或有差异 | 唯一 |
| **进入壁垒** | 无 | 低 | 高 | 极高 |
| **定价能力** | 价格接受者 | 有限 | 相互依赖 | 价格制定者 |
| **长期利润** | 零 | 零 | 可能正 | 可能正 |
| **效率** | 有效 | 过剩产能 | 不确定 | 无效 |

## 二、数理/逻辑推导

### 2.1 完全竞争企业供给曲线

**短期供给曲线**：
$$P = MC(Q), \quad P \geq AVC_{min}$$

**停止营业点**：$P = AVC_{min}$

**退出市场点**：$P = ATC_{min}$

### 2.2 垄断定价

**需求曲线**：$P = P(Q)$，向右下方倾斜

**边际收益**：
$$MR = P + Q \cdot \frac{dP}{dQ} = P\left(1 + \frac{1}{E_d}\right)$$

**垄断定价**：
$$P = \frac{MC}{1 + \frac{1}{E_d}}$$

**勒纳指数（Lerner Index）**：
$$L = \frac{P - MC}{P} = -\frac{1}{E_d}$$

### 2.3 价格歧视

**一级价格歧视（完全）**：
- 每单位产品按消费者最高支付意愿定价
- 消费者剩余 = 0
- 无谓损失 = 0

**二级价格歧视**：
- 按购买数量定价
- 如批量折扣

**三级价格歧视**：
- 按消费者群体定价
- $MR_1 = MR_2 = MC$

## 三、学术批判

### 3.1 垄断的社会福利损失

**无谓损失来源**：
- 垄断产量 $Q_m < Q_c$（竞争产量）
- 垄断价格 $P_m > MC$
- 边际买者的支付意愿 > 边际成本，但交易未发生

**DWL计算**：
$$DWL = \frac{1}{2}(P_m - MC)(Q_c - Q_m)$$

### 3.2 垄断竞争的特征

**短期**：可能盈利或亏损

**长期**：
- 自由进入 → 需求曲线左移
- 零经济利润
- $P = ATC$，但 $P > MC$
- 过剩产能（Excess Capacity）

### 3.3 寡头博弈

**纳什均衡**：
- 给定对方策略，各方策略最优
- 个体理性 → 可能集体非理性

**囚徒困境**：
| | 合作 | 背叛 |
|---|---|---|
| **合作** | (-1, -1) | (-10, 0) |
| **背叛** | (0, -10) | (-5, -5) |

**卡特尔的不稳定性**：
- 协议产量 < 背叛产量
- 个体激励导致协议破裂
