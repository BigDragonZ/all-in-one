## 一、核心定义

### 1.1 外部性（Externality）

**定义**：一个人的行为对旁观者福利的影响，而这种影响未体现在市场价格中。

**负外部性（Negative Externality）**：
$$\text{社会成本} = \text{私人成本} + \text{外部成本}$$

**正外部性（Positive Externality）**：
$$\text{社会价值} = \text{私人价值} + \text{外部收益}$$

### 1.2 科斯定理（Coase Theorem）

**命题**：如果产权界定清晰且交易成本为零，无论初始产权如何分配，私人谈判总能达成有效率的资源配置。

**条件**：
1. 产权清晰
2. 交易成本为零（或极低）

### 1.3 物品分类矩阵

| | 排他性 | 非排他性 |
|---|---|---|
| **竞争性** | 私人物品 | 公有资源 |
| **非竞争性** | 自然垄断 | 公共物品 |

## 二、数理/逻辑推导

### 2.1 外部性导致的市场失灵

**负外部性（如污染）**：
- 私人均衡：$P = MC_{私人}$
- 社会最优：$P = MC_{社会} = MC_{私人} + MC_{外部}$
- 结果：市场产量 $Q_{市场} > Q_{社会最优}$

**正外部性（如教育）**：
- 私人均衡：$P = MB_{私人}$
- 社会最优：$P = MB_{社会} = MB_{私人} + MB_{外部}$
- 结果：市场产量 $Q_{市场} < Q_{社会最优}$

### 2.2 庇古税与补贴

**庇古税（Pigouvian Tax）**：
- 对负外部性征税，税额 = 边际外部成本
- 使 $MC_{私人} + t = MC_{社会}$

**庇古补贴（Pigouvian Subsidy）**：
- 对正外部性补贴，补贴额 = 边际外部收益
- 使 $MB_{私人} + s = MB_{社会}$

### 2.3 公地悲剧（Tragedy of the Commons）

**机制**：
- 公有资源：非排他性 + 竞争性
- 个体理性：最大化个人收益
- 集体结果：资源过度使用直至枯竭

**数学表达**：
- 个体边际收益 $MR_{私人} > MC_{私人}$
- 社会边际成本 $MC_{社会} = MC_{私人} + MC_{外部}$
- 均衡时：$MR_{私人} = MC_{私人} < MC_{社会}$

## 三、学术批判

### 3.1 科斯定理的现实局限

**交易成本高昂**：
- 谈判成本
- 信息不对称
- 协调困难（利益相关者众多）

**产权界定困难**：
- 空气、海洋等难以界定产权
- 历史遗留问题

### 3.2 公共物品的供给

**搭便车问题（Free-rider Problem）**：
- 非排他性 → 无法排除未付费者
- 非竞争性 → 边际成本为零
- 结果：市场供给不足，需政府提供

**最优供给量**：
$$\sum_{i=1}^{n} MB_i = MC$$

### 3.3 政府干预的边界

**政府失灵（Government Failure）**：
- 信息不完全
- 官僚效率低下
- 寻租行为
- 政策时滞

**权衡**：
- 市场失灵 ≠ 政府必然能改善
- 需要成本-收益分析
