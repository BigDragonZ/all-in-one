## 一、核心定义

### 1.1 经济学十大原理（曼昆）

**人们如何做出决策**：
1. 人们面临权衡取舍（Trade-offs）
2. 某种东西的成本是为了得到它所放弃的东西（机会成本）
3. 理性人考虑边际量（Margin）
4. 人们会对激励做出反应

**人们如何相互交易**：
5. 贸易能使每个人状况更好
6. 市场通常是组织经济活动的一种好方法
7. 政府有时可以改善市场结果

**整体经济如何运行**：
8. 一国的生活水平取决于它生产物品与服务的能力
9. 当政府发行了过多货币时，物价上升
10. 社会面临通货膨胀与失业之间的短期权衡取舍

### 1.2 比较优势（Comparative Advantage）

**定义**：生产者以比另一生产者更低的机会成本生产某种物品的能力。

**数学表达**：
设两国生产两种商品 X 和 Y：
- 国家 A 生产 1 单位 X 的机会成本 = $a_Y$ 单位 Y
- 国家 B 生产 1 单位 X 的机会成本 = $b_Y$ 单位 Y

若 $a_Y < b_Y$，则 A 在 X 生产上具有比较优势。

**贸易条件**：交换比例介于两国机会成本之间
$$a_Y < \text{贸易比率} < b_Y$$

### 1.3 生产可能性边界（PPF）

**定义**：在既定资源和技术条件下，经济所能生产的两种商品最大组合的轨迹。

**斜率**：
$$\text{斜率} = -\frac{\Delta Y}{\Delta X} = \text{生产 X 的机会成本}$$

**性质**：
- 凹向原点（向外凸出）：反映边际收益递减
- 边界上的点：有效率
- 边界内的点：无效率
- 边界外的点：不可行

## 二、数理/逻辑推导

### 2.1 贸易增益的数学证明

**假设**：
- 李逵：1 斤鱼 = 10 斤柴（机会成本）
- 李鬼：1 斤鱼 = 5 斤柴（机会成本）

**比较优势判定**：
- 李鬼生产鱼的机会成本更低 → 李鬼在产鱼上有比较优势
- 李逵生产柴的机会成本更低 → 李逵在产柴上有比较优势

**专业化前**：
- 李逵生产 5 斤鱼 + 50 斤柴
- 李鬼生产 3 斤鱼 + 15 斤柴
- 总产量：8 斤鱼 + 65 斤柴

**专业化后**：
- 李鬼专攻鱼：8 斤鱼 + 0 斤柴
- 李逵专攻柴：0 斤鱼 + 100 斤柴
- 总产量：8 斤鱼 + 100 斤柴

**贸易增益**：柴增加 35 斤（100 - 65）

### 2.2 PPF 与机会成本

**线性 PPF**（机会成本不变）：
$$Y = a - bX$$

**非线性 PPF**（机会成本递增）：
$$Y = f(X), \quad f'(X) < 0, \quad f''(X) < 0$$

**技术进步的影响**：
- X 部门技术进步 → PPF 沿 X 轴外扩
- 整体技术进步 → PPF 整体外移

## 三、学术批判

### 3.1 实证分析与规范分析

| 维度 | 实证分析（Positive） | 规范分析（Normative） |
|------|---------------------|----------------------|
| **本质** | 描述世界"是什么" |  prescribe 世界"应该是什么" |
| **可验证性** | 可检验 | 依赖价值判断 |
| **例子** | "提高最低工资会增加失业" | "政府应该提高最低工资" |

### 3.2 循环流向图

**两部门模型**：
- 家庭：拥有生产要素，购买商品与服务
- 企业：雇佣生产要素，生产商品与服务

**两个市场**：
- 要素市场：家庭卖、企业买
- 商品市场：企业卖、家庭买

### 3.3 贸易的误解

**常见谬误**：
- "贸易是零和博弈" → 错误，贸易是双赢
- "绝对优势决定贸易" → 错误，比较优势决定贸易
- "自给自足更安全" → 忽略了专业化带来的效率增益
