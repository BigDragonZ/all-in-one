## 一、核心定义

### 1.1 开放经济恒等式

**国民收入恒等式**：
$$Y = C + I + G + NX$$

**储蓄-投资关系**：
$$S = I + NCO$$

其中 $NCO$ 为资本净流出（Net Capital Outflow）。

**净出口与资本净流出**：
$$NX = NCO$$

### 1.2 汇率

**名义汇率（e）**：
- 直接标价法：1单位外币 = 多少本币
- 间接标价法：1单位本币 = 多少外币

**实际汇率（ε）**：
$$\varepsilon = e \times \frac{P_{domestic}}{P_{foreign}}$$

其中：
- $e$：名义汇率（间接标价法）
- $P_{domestic}$：国内物价水平
- $P_{foreign}$：国外物价水平

### 1.3 购买力平价（PPP）

**一价定律**：
$$P_{domestic} = e \times P_{foreign}$$

**实际汇率**：
$$\varepsilon = 1$$

## 二、数理/逻辑推导

### 2.1 开放经济均衡模型

**可贷资金市场**：
- 供给：国民储蓄 $S$
- 需求：国内投资 $I$ + 资本净流出 $NCO$
- 均衡：真实利率 $r$ 使 $S = I + NCO$

**外汇市场**：
- 供给：资本净流出 $NCO$（本币供给）
- 需求：净出口 $NX$（本币需求）
- 均衡：实际汇率 $\varepsilon$ 使 $NCO = NX$

**联动机制**：
1. 利率 $r$ → 影响 $NCO$
2. $NCO$ → 外汇市场本币供给
3. 本币供给与需求 → 实际汇率 $\varepsilon$

### 2.2 贸易政策的影响

**关税**：
- 国内价格上升
- 进口减少
- 净出口增加（短期）
- 但 $NX = NCO$ 恒等式约束
- 实际汇率上升 → 出口竞争力下降
- 最终 $NX$ 不变

**结论**：
- 贸易政策不影响贸易余额（$NX$）
- 只影响实际汇率和贸易结构

### 2.3 资本流动

**资本流入**：
- 外国购买本国资产
- $NCO < 0$（资本净流入）
- 本币需求增加 → 汇率升值

**资本流出**：
- 本国购买外国资产
- $NCO > 0$（资本净流出）
- 本币供给增加 → 汇率贬值

## 三、学术批判

### 3.1 购买力平价的局限

**短期偏离**：
- 非贸易品存在
- 贸易壁垒
- 运输成本

**长期趋势**：
- 高通胀国家货币贬值
- 符合PPP预测

### 3.2 汇率制度

**固定汇率**：
- 央行干预外汇市场
- 牺牲货币政策独立性

**浮动汇率**：
- 汇率由市场决定
- 货币政策独立

**三元悖论（Impossible Trinity）**：
- 固定汇率
- 资本自由流动
- 货币政策独立
- 三者不可兼得

### 3.3 国际收支

**经常账户**：
- 贸易余额（$NX$）
- 净要素收入

**资本账户**：
- 资本净流出（$NCO$）

**平衡**：
$$CA + KA = 0$$
