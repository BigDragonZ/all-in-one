## 一、核心定义

### 1.1 总需求-总供给模型

**总需求（AD）**：
$$Y = C + I + G + NX$$

**AD曲线向下倾斜的原因**：
1. 财富效应（$P \downarrow \rightarrow$ 实际财富 $\uparrow \rightarrow C \uparrow$）
2. 利率效应（$P \downarrow \rightarrow$ 实际货币供给 $\uparrow \rightarrow r \downarrow \rightarrow I \uparrow$）
3. 汇率效应（$P \downarrow \rightarrow r \downarrow \rightarrow$ 本币贬值 $\rightarrow NX \uparrow$）

**短期总供给（SRAS）**：
$$Y = Y_N + a(P - P_E)$$

向上倾斜的原因：粘性工资、粘性价格、错觉理论。

**长期总供给（LRAS）**：
$$Y = Y_N$$

垂直的原因：货币中性，产出由生产要素决定。

### 1.2 流动性偏好理论

**货币需求**：
$$\frac{M^d}{P} = L(r, Y)$$

向下倾斜：利率越高，持币机会成本越高。

**货币供给**：
$$\frac{M^s}{P} = \bar{M}$$

由央行决定。

**均衡利率**：
$$L(r, Y) = \frac{\bar{M}}{P}$$

### 1.3 菲利普斯曲线

**短期菲利普斯曲线**：
$$\pi = \pi^e - \beta(u - u_n) + \nu$$

其中：
- $\pi$：实际通胀率
- $\pi^e$：预期通胀率
- $u$：实际失业率
- $u_n$：自然失业率
- $\nu$：供给冲击

**长期菲利普斯曲线**：
$$u = u_n$$

垂直于自然失业率。

## 二、数理/逻辑推导

### 2.1 乘数效应

**消费函数**：
$$C = C_0 + c(Y - T)$$

其中 $c$ 为边际消费倾向（MPC）。

**均衡产出**：
$$Y = \frac{1}{1-c}(C_0 + I + G - cT)$$

**政府购买乘数**：
$$\frac{\Delta Y}{\Delta G} = \frac{1}{1-c}$$

**税收乘数**：
$$\frac{\Delta Y}{\Delta T} = -\frac{c}{1-c}$$

### 2.2 货币政策传导

**扩张性货币政策**：
1. 央行增加货币供给 $M \uparrow$
2. 利率下降 $r \downarrow$
3. 投资增加 $I \uparrow$
4. 总需求增加 $AD \rightarrow$ 右移
5. 短期产出增加 $Y \uparrow$
6. 长期物价上升 $P \uparrow$

### 2.3 财政政策传导

**扩张性财政政策**：
1. 政府购买增加 $G \uparrow$ 或税收减少 $T \downarrow$
2. 总需求增加 $AD \rightarrow$ 右移
3. 短期产出增加 $Y \uparrow$
4. 利率上升（挤出效应）
5. 投资部分被挤出

**挤出效应**：
$$\text{完全挤出} \rightarrow \Delta Y = 0$$

## 三、学术批判

### 3.1 滞胀（Stagflation）

**定义**：高通胀与高失业并存。

**原因**：
- 不利的供给冲击（如石油危机）
- $SRAS$ 左移
- 产出下降 + 物价上升

**政策困境**：
- 扩张政策 → 通胀恶化
- 紧缩政策 → 失业加剧

### 3.2 理性预期

**卢卡斯批判**：
- 传统政策评估忽略预期调整
- 若公众预期到政策，政策效果消失

**政策含义**：
- 系统性货币政策无效
- 只有未预期的政策有影响

### 3.3 时间不一致性

**基德兰德-普雷斯科特**：
- 政策制定者有事后偏离承诺的激励
- 导致通胀倾向

**解决方案**：
- 央行独立性
- 规则优于相机抉择
