## 一、核心定义

### 1.1 需求价格弹性

**弧弹性（中点法）**：
$$E_d = \frac{(Q_2 - Q_1) / [(Q_2 + Q_1)/2]}{(P_2 - P_1) / [(P_2 + P_1)/2]}$$

**点弹性**：
$$E_d = \frac{dQ}{dP} \cdot \frac{P}{Q}$$

**弹性分类**：
- $|E_d| > 1$：富有弹性
- $|E_d| = 1$：单位弹性
- $|E_d| < 1$：缺乏弹性
- $|E_d| = 0$：完全无弹性
- $|E_d| = \infty$：完全弹性

### 1.2 消费者剩余与生产者剩余

**消费者剩余（CS）**：
$$CS = \text{支付意愿} - \text{实际支付} = \int_0^{Q^*} D(Q) dQ - P^* Q^*$$

**生产者剩余（PS）**：
$$PS = \text{实际收入} - \text{生产成本} = P^* Q^* - \int_0^{Q^*} S(Q) dQ$$

**总剩余**：
$$TS = CS + PS$$

## 二、数理/逻辑推导

### 2.1 市场均衡

**均衡条件**：
$$Q_d(P) = Q_s(P)$$

**均衡价格 $P^*$ 和均衡数量 $Q^*$**：供求曲线的交点

**市场效率**：在均衡点，总剩余最大化

### 2.2 税收归宿

**税收楔子**：
$$P_b - P_s = t$$

其中 $P_b$ 为买方支付价格，$P_s$ 为卖方获得价格，$t$ 为单位税额。

**税收负担分配**：
- 买方负担：$P_b - P^*$
- 卖方负担：$P^* - P_s$

**弹性决定归宿**：
$$\frac{\text{买方负担}}{\text{卖方负担}} = \frac{E_s}{E_d}$$

**结论**：税收负担倾向于落在缺乏弹性的一方

### 2.3 价格管制

**价格上限（Price Ceiling）**：
- 设定 $P_{max} < P^*$ → 短缺
- 无谓损失：本可发生的交易被阻止

**价格下限（Price Floor）**：
- 设定 $P_{min} > P^*$ → 过剩
- 最低工资是典型的价格下限

## 三、学术批判

### 3.1 "谷贱伤农"现象

**机制**：
- 粮食需求缺乏弹性（必需品）
- 丰收 → 供给曲线右移
- 价格大幅下降（%ΔP 大）
- 需求量小幅增加（%ΔQ 小）
- 总收益 $TR = P \times Q$ 下降

**数学表达**：
$$\frac{\Delta TR}{TR} \approx \frac{\Delta P}{P} + \frac{\Delta Q}{Q} = \frac{\Delta P}{P}(1 + E_d)$$

当 $|E_d| < 1$ 时，$\Delta TR$ 与 $\Delta P$ 同向变动

### 3.2 无谓损失（Deadweight Loss）

**来源**：
- 税收扭曲了边际决策
- 边际买者和边际卖者之间的交易被阻止

**计算**：
$$DWL = \frac{1}{2} \cdot t \cdot \Delta Q$$

其中 $t$ 为税额，$\Delta Q$ 为交易量减少

### 3.3 国际贸易中的剩余分析

**自由贸易**：
- 出口国：生产者剩余增加 > 消费者剩余减少
- 进口国：消费者剩余增加 > 生产者剩余减少

**关税**：
- 国内价格上升
- 消费者剩余减少
- 生产者剩余增加
- 政府获得关税收入
- 产生无谓损失
