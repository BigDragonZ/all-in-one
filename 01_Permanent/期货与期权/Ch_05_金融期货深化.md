## 一、核心定义

### 1.1 股指期货（Stock Index Futures）

以股票价格指数为标的资产的期货合约，采用现金结算。

**沪深300股指期货关键参数**：
- **合约乘数**：每点300元
- **保证金比例**：最低8%（杠杆倍数 = $1 \div 8\% = 12.5$倍）
- **结算方式**：现金结算
- **合约月份**：当月、下月及随后两个季月

**合约价值计算**：
$$V_{contract} = \text{指数点数} \times 300$$

### 1.2 国债期货（Treasury Bond Futures）

以国债为标的资产的期货合约，采用实物交割。

**关键概念**：
- **名义标准券**：虚拟的标准化债券（如票面利率6%、20年期）
- **可交割债券名单（List）**：实际可用于交割的真实债券集合
- **转换因子（Conversion Factor, CF）**：将真实债券折算为标准券的比例
- **最便宜可交割债券（CTD）**：交割成本最低的真实债券

### 1.3 欧洲美元期货（Eurodollar Futures）

以三个月期欧洲美元存款利率（Libor）为标的的利率期货。

**报价方式**：
$$\text{IMM指数} = 100 - R$$

其中 $R$ 为年化利率（%）。

**盈亏计算**：
- 每1个基点（0.01%）变动 = 25美元/张
- 计算逻辑：$1,000,000 \times 0.0001 \times 0.25 = 25$

## 二、数理/逻辑推导

### 2.1 转换因子（CF）计算

假设到期收益率为标准券票面利率（如6%），按半年复利计算可交割债券在交割日后的现值：

$$CF = \frac{1}{(1+r/2)^{n/6}} \left( \frac{C}{2} + \frac{C}{2} \sum_{i=1}^{m} \frac{1}{(1+r/2)^i} + \frac{100}{(1+r/2)^m} \right)$$

简化理解：
- 票面利率 > 标准利率 → CF > 1（生水）
- 票面利率 < 标准利率 → CF < 1（贴水）
- 标准券本身 CF = 1

### 2.2 CTD债券判定

**交割成本计算**：
$$\text{交割成本} = \text{现货报价} - (\text{期货结算价} \times CF)$$

**CTD判定**：选择交割成本最小的债券

**期货价格锚定**：
$$F \approx \frac{P_{CTD}}{CF_{CTD}}$$

### 2.3 Beta管理与Alpha保留

**Beta管理（对冲系统性风险）**：

$$N = \frac{V_{portfolio} \times \beta}{V_{futures}}$$

**Alpha保留策略**：
- 买入正Alpha个股 + 做空股指期货
- 对冲掉市场波动（Beta），保留个股超额收益（Alpha）
- 效果：大盘下跌时期货盈利抵消股票跌幅；个股强于大盘时赚取Alpha差额

### 2.4 杠杆效应

**杠杆倍数**：
$$\text{Leverage} = \frac{1}{\text{Margin Ratio}}$$

**收益/亏损放大**：
$$\text{实际收益率} = \text{标的资产变动率} \times \text{杠杆倍数}$$

示例：8%保证金 → 12.5倍杠杆
- 指数上涨1% → 保证金收益12.5%
- 指数下跌1% → 保证金亏损12.5%

## 三、学术批判

### 3.1 外汇期货与外汇远期对比

| 维度 | 外汇远期（Forward） | 外汇期货（Futures） |
|------|---------------------|---------------------|
| **交易场所** | OTC场外 | 交易所场内 |
| **合约设计** | 定制化 | 标准化（固定合约规模） |
| **结算方式** | 到期一次性结算 | 每日无负债结算 |
| **信用风险** | 高（双边信用） | 低（CCP担保） |
| **杠杆** | 无固定杠杆 | 保证金杠杆（如12.5倍） |
| **流动性** | 低 | 高 |

### 3.2 交叉汇率期货与三角套利

**定价约束**：
$$F_{EUR/JPY} = F_{EUR/USD} \times F_{USD/JPY}$$

若出现偏差：
- 买入低估的交叉汇率期货
- 卖出高估的直盘汇率期货组合
- 套利行为强制价格回归均衡

### 3.3 保证金动态调整机制

**提高保证金的情形**：
1. **节假日**：应对国际市场"跳空"风险
2. **市场波动剧烈**：防止系统性违约
3. **临近交割月**：逼迫投机头寸平仓

**经济逻辑**：
- 交易所作为CCP承担履约责任
- 提高保证金 = 增加账户"垫片"
- "和尚跑了庙还在"——防止大面积穿仓

### 3.4 中国内地市场特殊监管

| 监管特征 | 具体内容 |
|----------|----------|
| **交易所组织** | 中金所：公司制；其他三家：会员制 |
| **会员隔离** | 经纪会员禁止自营；自营会员禁止经纪 |
| **大户报告** | 达到规模需申报资金来源、交易动机、现货储备 |
| **散户交割限制** | 自然人客户禁止黄金期货实物交割 |
| **午休制度** | 全球罕见的午间休市 |
| **价格限制** | 以结算价为基准，交易所可动态调整幅度 |
| **投资者分级** | 期权市场一、二、三级管理，初级禁止裸卖 |
