## 一、核心定义

### 1.1 保护性看跌期权（Protective Put）

**构造**：多头股票 + 多头看跌期权

**到期收益**：
$$Payoff = S_T + \max(K - S_T, 0) = \max(S_T, K)$$

**功能**：为股票头寸提供下行保护，锁定最大亏损。

### 1.2 备兑看涨期权（Covered Call）

**构造**：多头股票 + 空头看涨期权

**到期收益**：
$$Payoff = S_T - \max(S_T - K, 0) = \min(S_T, K)$$

**功能**：通过放弃上行收益换取权利金收入。

**等价性**：Covered Call $\equiv$ Short Put

### 1.3 跨式组合（Straddle）

**构造**：买入相同执行价格 $K$ 和到期日 $T$ 的看涨和看跌期权

**到期收益**：
$$Payoff = \max(S_T - K, 0) + \max(K - S_T, 0) = |S_T - K|$$

**适用场景**：高波动率预期，不赌方向赌波动。

### 1.4 宽跨式组合（Strangle）

**构造**：买入不同执行价格（$K_1 < K_2$）的看跌和看涨期权

**特点**：成本低于跨式，但需要更大价格波动才能盈利。

## 二、数理/逻辑推导

### 2.1 牛市价差（Bull Spread）

**构造（看涨期权）**：
- 买入低执行价 $K_1$ 看涨期权
- 卖出高执行价 $K_2$ 看涨期权

**到期收益**：
$$Payoff = \max(S_T - K_1, 0) - \max(S_T - K_2, 0)$$

**分段函数**：
- $S_T \le K_1$：亏损 = 净权利金支出
- $K_1 < S_T < K_2$：盈利随 $S_T$ 线性增加
- $S_T \ge K_2$：盈利封顶 = $K_2 - K_1$ - 净权利金

### 2.2 蝶式价差（Butterfly Spread）

**构造（看涨期权）**：
- 买入1份 $K_1$ 看涨
- 卖出2份 $K_2$ 看涨
- 买入1份 $K_3$ 看涨

其中 $K_1 < K_2 < K_3$，通常 $K_2 = \frac{K_1 + K_3}{2}$

**收益特征**：
- 在 $K_2$ 附近达到最大收益
- 股价大幅偏离 $K_2$ 时产生有限亏损
- 适合盘整行情

### 2.3 日历价差（Calendar Spread）

**构造**：
- 卖出短期限期权（到期 $T_1$）
- 买入长期限期权（到期 $T_2 > T_1$）
- 相同执行价格 $K$

**获利原理**：
- 短期限期权Theta衰减更快
- 股价接近 $K$ 时，短期期权价值迅速损耗
- 长期期权价值衰减较慢
- 价差扩大即为盈利

### 2.4 盒子价差（Box Spread）

**构造**：牛市看涨价差 + 熊市看跌价差

**特性**：
- 在欧式期权下等价于无风险资产
- 价值锁定：$(K_2 - K_1)e^{-rT}$
- 若市场价格偏离，存在套利机会

## 三、学术批判

### 3.1 希腊字母协同管理

| 希腊字母 | 含义 | 管理目标 |
|----------|------|----------|
| **Delta ($\Delta$)** | 价格敏感度 | Delta中性：规避方向性风险 |
| **Gamma ($\Gamma$)** | Delta变化率 | 正Gamma：捕捉波动收益 |
| **Theta ($\Theta$)** | 时间衰减 | 收取时间价值（卖方） |
| **Vega ($V$)** | 波动率敏感度 | 波动率交易核心指标 |

**协同关系**：
- Gamma与Theta互为对手：正Gamma通常伴随负Theta
- Delta中性 + 正Gamma = 波动率交易经典组合
- 做市商：Delta中性 + 收取Theta + 管理Gamma

### 3.2 风险偏好光谱

| 策略类型 | 代表策略 | 风险偏好 | 核心目标 |
|----------|----------|----------|----------|
| **保险型** | 保护性看跌 | 保守防御 | 下行保护 |
| **投机型** | 牛市/熊市价差 | 方向性预测 | 有限风险收益 |
| **波动率型** | 跨式/宽跨式 | 捕捉不稳定性 | 波动率交易 |
| **收益增强型** | 备兑看涨 | 温和看涨 | 权利金收入 |
| **盘整型** | 蝶式/日历价差 | 低波动预期 | 时间价值/范围获利 |

### 3.3 初学者常见错误

1. **绘图比例错误**：执行价格与盈亏轴长度不一致
2. **忽略Theta**：不理解时间价值非线性衰减
3. **杠杆幻觉**：只看到高收益，忽略同比例高风险
4. **裸卖风险**：未意识到理论上无限亏损
5. **合规忽视**：内地一级投资者必须背对开仓

### 3.4 期权作为"金融乐高"

**模块化特征**：
- 基础组件：Call/Put × Long/Short × K × T
- 非线性合成：精确切掉不想要的风险
- 按需定制：匹配任意市场预期

**合成关系示例**：
- Covered Call $\equiv$ Short Put
- Protective Put + Short Call $\equiv$ Collar
- Box Spread $\equiv$ 无风险资产
