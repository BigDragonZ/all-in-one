## 一、核心定义

### 1.1 无套利原理（No-Arbitrage Principle）

在有效市场中，不存在零成本、零风险且正收益的投资机会。

**一价定律（Law of One Price）**：
若两个资产在未来所有状态下的现金流完全相同，则当前价格必须相等。

### 1.2 看涨期权价格边界

**上界**：
$$c \le S_0, \quad C \le S_0$$

**下界（欧式）**：
$$c \ge \max(S_0 - Ke^{-rT}, 0)$$

### 1.3 看跌期权价格边界

**上界**：
- 欧式：$p \le Ke^{-rT}$
- 美式：$P \le K$

**下界（欧式）**：
$$p \ge \max(Ke^{-rT} - S_0, 0)$$

### 1.4 看跌-看涨平价关系（Put-Call Parity）

$$c + Ke^{-rT} = p + S_0$$

## 二、数理/逻辑推导

### 2.1 看涨期权上界推导（反证法）

**假设**：$C > S_0$

**套利策略**：
1. 卖出看涨期权（收入 $C$）
2. 买入股票（支出 $S_0$）
3. 期初现金流：$C - S_0 > 0$

**到期结果**：
- 若 $S_T > K$：买方行权，交出股票获得 $K$。总利润 = $(C - S_0) + K > 0$
- 若 $S_T \le K$：买方不行权，持有股票。总利润 = $(C - S_0) + S_T > 0$

**结论**：假设不成立，$C \le S_0$。

### 2.2 看涨期权下界推导（组合比较法）

**构造组合**：
- **组合A**：1份欧式看涨期权 + 现金 $Ke^{-rT}$
- **组合B**：1份标的股票

**到期价值**：
- 组合A：$\max(S_T - K, 0) + K = \max(S_T, K)$
- 组合B：$S_T$

**比较**：$\max(S_T, K) \ge S_T$

**无套利条件**：
$$c + Ke^{-rT} \ge S_0$$

$$c \ge S_0 - Ke^{-rT}$$

结合非负约束：$c \ge \max(S_0 - Ke^{-rT}, 0)$

### 2.3 看跌-看涨平价推导

**构造组合**：
- **组合A**：1份欧式看涨期权 + 现金 $Ke^{-rT}$
- **组合C**：1份欧式看跌期权 + 1份标的股票

**到期价值**：
- 组合A：$\max(S_T - K, 0) + K = \max(S_T, K)$
- 组合C：$\max(K - S_T, 0) + S_T = \max(S_T, K)$

**无套利条件**：
$$c + Ke^{-rT} = p + S_0$$

### 2.4 期权价格影响因素

| 因素 | 看涨期权 | 看跌期权 |
|------|----------|----------|
| **标的资产价格 $S$** | + | - |
| **执行价格 $K$** | - | + |
| **到期时间 $T$（美式）** | + | + |
| **到期时间 $T$（欧式）** | +（无红利） | 不确定 |
| **波动率 $\sigma$** | + | + |
| **无风险利率 $r$** | + | - |

### 2.5 隐含波动率（Implied Volatility）

**定义**：将期权市场价格代入定价模型，反推得到的波动率。

**计算**：
$$\sigma_{IV} = BS^{-1}(V_{market}, S, K, T, r)$$

**市场情绪指标**：
- IV上升 → 市场恐慌、预期剧烈波动
- IV下降 → 市场平静、预期稳定

## 三、学术批判

### 3.1 Bachelier vs Black-Scholes-Merton

| 维度 | Bachelier (1900) | Black-Scholes-Merton (1973) |
|------|------------------|------------------------------|
| **均衡范式** | 一般均衡 | 无套利均衡 |
| **核心方法** | 统计描述 | 动态复制组合 |
| **独立性** | 依附于经济学 | 金融学自立山头 |
| **历史意义** | 早期探索 | 期权定价革命 |

### 3.2 BSM模型的革命性意义

1. **理论定价空间**：为期权价格设定严格理论边界
2. **复制技术**：通过股票+无风险资产动态复制期权支付
3. **脱离偏好依赖**：定价不依赖投资者风险偏好
4. **与CBOE同年诞生**：1973年标志期权进入规范场内交易时代

### 3.3 波动率微笑与BSM局限性

**波动率微笑（Volatility Smile）**：
- 同一标的、不同执行价的期权隐含波动率不同
- 深度实值/虚值期权的IV高于平值期权

**波动率偏斜（Volatility Skew）**：
- 看跌期权的IV高于看涨期权（暴跌恐慌溢价）
- 反映了市场对极端下行风险的非对称预期

**BSM局限性**：
- 假设波动率为常数 → 与微笑现象矛盾
- 假设标的资产服从对数正态分布 → 无法解释肥尾现象
- 无法解释不同行权价的IV差异

### 3.4 波动率的"浅变量"属性

**不可观测性**：
- $S, K, T, r$ 均可直接观测
- $\sigma$ 隐藏在价格运动背后，无法直接测量
- 需要历史数据估计或市场价格反推

**预测挑战**：
- 定价需要未来波动率
- 只能观测过去走势
- 历史波动率 ≠ 未来波动率
